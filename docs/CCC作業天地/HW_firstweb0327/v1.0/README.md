# v1.0 — 網頁日誌系統（初版）

> 建立日期：2026-03-27

---

## 📋 版本說明

第一版交付。實作基本功能，沒有繁複的UI，但涵蓋所有核心需求。

---

## ✨ 本版功能

| 功能 | 說明 |
|------|------|
| ➕ 新增日誌 | 填寫標題、日期、內容後儲存 |
| ✏️ 編輯 | 點選日誌右側編輯按鈕修改 |
| 🗑️ 刪除 | 刪除單篇日誌 |
| 🔍 搜尋 | 即時過濾標題與內容 |
| 📂 展開/收合 | 點擊日誌卡片展開內容 |
| 💾 匯出 / 📂 匯入 | 備份與還原 SQLite 資料庫 |
| 🗑️ 清除全部 | 一鍵清除所有日誌 |

---

## 🔧 技術架構

```
index.html  ← HTML 結構
journal.js  ← 所有 JS 邏輯
  ├── sql.js (CDN)         ← SQLite 引擎（WASM）
  ├── IndexedDB 持久化      ← 關閉視窗資料不消失
  └── DOM 操作              ← Render / Event
```

---

## 📁 檔案

- `index.html` — 日誌系統主頁
- `journal.js` — 所有邏輯

---

## 🚀 使用方式

需透過本機 HTTP 伺服器啟動：

```bash
# 在 v1.0/ 的上層資料夾執行
python3 -m http.server 8080
# 然後打開 http://localhost:8080/v1.0/index.html
```
