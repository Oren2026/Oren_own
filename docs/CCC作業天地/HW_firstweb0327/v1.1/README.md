# v1.1 — 網頁日誌系統（使用者系統）

> 建立日期：2026-03-27｜從 v1.0 更新

---

## 📋 版本說明

在 v1.0 基礎上新增**使用者註冊與登入系統**，每位使用者的日誌各自獨立。

---

## ✨ 本版新增功能

| 功能 | 說明 |
|------|------|
| 👤 註冊 | 填寫帳號 + 密碼（SHA-256 雜湊儲存） |
| 🔐 登入 | 帳密驗證後進入個人日誌 |
| 🚪 登出 | 結束當前 session |
| 👥 多使用者 | 各人日誌資料隔離，不互通 |

---

## 🔧 技術變更

- **資料庫結構更新**：
  - `users` 資料表：儲存帳號與密碼雜湊（`password_hash`）
  - `entries` 資料表：新增 `user_id` 關聯
- **密碼處理**：密碼經 SHA-256 雜湊後儲存（不儲存明文）
- **Session 管理**：登入狀態存於 `sessionStorage`，關閉瀏覽器後自動登出
- **匯入匯出**：匯入後需重新登入（舊 session 失效）

---

## 📁 檔案

- `index.html` — 日誌系統主頁（已整合登入 UI）
- `journal.js` — 所有邏輯（已新增 auth 功能）

---

## 🚀 使用方式

```bash
# 在 HW_firstweb0327/ 的上層執行：
python3 -m http.server 8080
# 然後打開 http://localhost:8080/HW_firstweb0327/v1.1/index.html
```

---

## 🔐 安全性說明

- 密碼使用 **SHA-256 雜湊**儲存，資料庫中不儲存明文密碼
- 各使用者的日誌透過 `user_id` 完全隔離
- Session 存在瀏覽器 `sessionStorage`，關閉視窗後失效
- 匯出的 `.db` 檔包含所有使用者資料，還原後需重新登入
