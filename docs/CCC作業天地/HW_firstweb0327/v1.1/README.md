# v1.1 — 網頁日誌系統

> 建立日期：2026-03-27｜初始正式版

---

## 📋 版本說明

第一個正式版本，公開日誌牆 + Auth Modal。

---

## ✨ 功能

| 功能 | 說明 |
|------|------|
| 🌍 公開日誌牆 | 所有人的文章都在首頁 |
| 🔐 Auth Modal | 雙面板式登入/註冊 |
| ➕ 發文 | 登入後可發表、編輯、刪除自己的文章 |
| 🔍 搜尋 | 即時過濾標題與內容 |
| 💾 匯出/匯入 | SQLite 資料庫備份與還原 |
| 🔐 SHA-256 | 密碼保護 |
