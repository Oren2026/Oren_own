# v1.1 — 網頁日誌系統（重構版）

> 建立日期：2026-03-27｜從 v1.0 重大更新

---

## 📋 版本說明

重新設計系統架構：日から全使用者共用的「公開日誌牆」，右上角有登入/註冊按鈕。

---

## ✨ 本版功能

| 功能 | 說明 |
|------|------|
| 🌍 公開日誌牆 | 所有人的文章都在首頁，依時間排序 |
| 👤 作者標記 | 每篇文章顯示作者名稱 |
| 🔐 Auth Modal | 登入/註冊變成浮動視窗，不跳頁 |
| ➕ 發表文章 | 登入後可發表、編輯、刪除自己的文章 |
| 🔍 搜尋 | 即時過濾標題與內容 |
| 💾 匯出 / 📂 匯入 | SQLite 資料庫備份與還原 |
