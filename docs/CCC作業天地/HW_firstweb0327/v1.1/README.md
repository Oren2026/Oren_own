# v1.2 — 網頁日誌系統

> 更新日期：2026-03-27｜從 v1.0 / v1.1 演進

---

## 📋 版本說明

**v1.2 新功能：**
- ✅ 點擊作者名稱 → 只看該作者的文章
- ✅ 右上角 filter chip 可一鍵清除，恢復全顯示
- ✅ 所有文章仍為公開顯示，任何人都能看到

---

## ✨ 功能一覽

| 功能 | 說明 |
|------|------|
| 👤 作者過濾 | 點作者名稱只看該作者，右上角 ✕ 清除 |
| 🌍 公開日誌牆 | 所有人的文章都在首頁 |
| 🔐 Auth Modal | 浮動面板式登入/註冊 |
| ➕ 發文 | 登入後可發表、編輯、刪除自己的文章 |
| 🔍 搜尋 | 即時過濾標題與內容 |
| 💾 匯出/匯入 | SQLite 資料庫備份與還原 |
| 🔐 SHA-256 | 密碼保護 |

## 🚀 使用方式

**必須在 HTTP 環境開啟**（seed.db 需 fetch）：

```bash
# 在 HW_firstweb0327/ 的上層執行：
python3 -m http.server 8080
# 然後打開 http://localhost:8080/HW_firstweb0327/v1.1/index.html
```
