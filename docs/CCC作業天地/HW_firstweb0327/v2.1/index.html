版本對照
🚀 v2.1 — Node.js 後端版（本頁）
  • SQLite 跑在 server，不是你電腦的瀏覽器
  • 密碼用 bcrypt，hash 在後端處理
  • JWT + cookie 管理登入，無法被 JS 偷走密碼
  • 真正的 REST API，前後端完全分離
  • 可擴展，輕鬆加密、多人同步、真實用戶系統
  • 需要在本機執行 node 服務
⚡ v1.x — 純前端版（sql.js / IndexedDB）
  • SQLite 在瀏覽器跑（sql.js / WASM）
  • 密碼用 SHA-256，前端 hash，安全性較低
  • IndexedDB 存本地，無伺服器依賴
  • GitHub Pages 直接托管，零成本
  • 適合展示，教學用、小工具
  • 所有人的資料存在自己的瀏覽器裡
