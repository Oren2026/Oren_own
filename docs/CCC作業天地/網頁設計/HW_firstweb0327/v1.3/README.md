# v1.3 — 網頁日誌系統

> 更新日期：2026-03-27｜從 v1.2 重構框架

---

## 📋 版本說明

從 v1.2 大幅重構 SQLite 邏輯框架，安全性與可維護性提升。

---

## 🔧 v1.3 框架改進

| 項目 | v1.2 | v1.3 |
|------|------|------|
| SQL 查詢 | 字串拼接（SQL 注入風險） | 參數化查詢（`db.get()` / `db.all()` / `db.run()`） |
| DB 封裝 | 無，直接操作全域 `db` | `DB class` 封裝所有 SQL 操作 |
| 參數傳遞 | 多處 `replace(/'/g,"''")` | 統一參數化，零拼接 |
| `persist()` | 有地方沒 await | 改為同步 fire-and-forget（sql.js export 本身是同步的） |
| IndexedDB key | `journal_sqlite_v2` | `journal_sqlite_v3`（避開舊版殘留） |
| Session key | `journal_session_v2` | `journal_session_v3` |

---

## ✨ 完整功能

| 功能 | 說明 |
|------|------|
| 👤 **作者過濾** | 點擊作者名稱 → 只看該作者文章，右上角 ✕ 清除 |
| 🌍 公開日誌牆 | 所有人的文章都顯示 |
| 🔐 Auth Modal | 浮動雙面板登入/註冊 |
| ➕ 發文 | 登入後可發表、編輯、刪除自己的文章 |
| 🔍 搜尋 | 即時過濾標題與內容 |
| 💾 匯出/匯入 | SQLite 資料庫備份與還原 |
