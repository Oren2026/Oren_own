# 系統架構規劃

## 節點架構

```
core_node_controller（主控制器）
  ├── roslaunch 動態啟閉各 detect_* 節點
  ├── 訂閱 /mission（狀態報告）
  └── 任務混合層（core_node_mission）— 規劃中

各偵測節點（detect_*.py）
  ├── 讀取 Camera Image
  ├── 輸出 /detect/xxx（各自 topic）
  └── 由 core_node_controller 統一調度
```

## Topic 流向

```
┌─────────────┐     Image      ┌──────────────────┐
│  Camera    │ ──────────────▶│  detect_*.py     │
│  (USB Cam) │                │                  │
└─────────────┘                │  輸出: /detect/* │
                               └────────┬─────────┘
                                        │ 訂閱
                               ┌────────▼─────────┐
                               │ core_node_*      │
                               │ （控制器）        │
                               └────────┬─────────┘
                                        │ cmd_vel
                               ┌────────▼─────────┐
                               │ /cmd_vel         │
                               │ （速度控制）      │
                               └──────────────────┘
```
