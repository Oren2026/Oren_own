## 個別節點測試

### 🚦 1. detect_traffic_light（交通號誌）
```bash
# 正常模式
roslaunch detect detect_traffic_light.launch

# 校正模式（調整 HSV 參數）
roslaunch detect detect_traffic_light.launch mode:=calibration
```

**發佈:** `/mission` (UInt8), `/cmd_vel` (Twist)
**訂閱:** `/image_input` (/camera/image/compressed)
**YAML:** `detect/yaml/traffic_light.yaml`
**校正输出:** `/detect/traffic_light/image_red/compressed`, `.../image_yellow/compressed`, `.../image_green/compressed`

**手動測試訊號:**
```bash
# 模擬綠燈 → 進 S_Curve
rostopic pub /mission std_msgs/UInt8 "data: 2"

# 模擬紅燈 → 停車
rostopic pub /cmd_vel geometry_msgs/Twist "linear: {x: 0.0}" "angular: {z: 0.0}"
```

### 🌀 2. detect_intersection（岔路/圓環）
```bash
roslaunch detect detect_intersection.launch
```

**發佈:** `/mission` (UInt8), `/control/lane` (Float64), `/cmd_vel` (Twist)
**訂閱:** `/image_input`, `/odom`
**參考圖:** `detect/image/intersection.png`, `left.png`, `right.png`
**關鍵參數:** `MIN_MATCH_COUNT=9`, `MIN_MSE_DECISION=50000`

**手動測試:**
```bash
# 看 intersection 辨識結果
rostopic echo /detect/sign/compressed

# 模擬左轉
rostopic pub /mission std_msgs/UInt8 "data: 3"
```

---

### 🚧 3. detect_construction（施工區/障礙）
```bash
roslaunch detect detect_construction.launch
```

**發佈:** `/mission` (UInt8), `/control/lane` (Float64)
**訂閱:** `/image_input`, `/scan` (LaserScan)
**參考圖:** `detect/image/construction.png`
**LiDAR 障礙偵測:** 前方 5° 範圍內 0.35~0.4m 有障礙物

**手動測試:**
```bash
# 看障礙偵測是否觸發
rostopic echo /mission

# 模擬進下一關
rostopic pub /mission std_msgs/UInt8 "data: 4"
```

### 🅿️ 4. detect_parking（停車場）
```bash
roslaunch detect detect_parking.launch
```

**發佈:** `/mission` (UInt8), `/control/moving/state` (MovingParam), `/cmd_vel` (Twist)
**訂閱:** `/image_input`, `/control/moving/complete`, `/scan` (LaserScan)
**參考圖:** `detect/image/parking.png`
**障礙偵測:** 左右各 45° 範圍內 0.2~0.3m

**手動測試:**
```bash
# 模擬進停車
rostopic pub /mission std_msgs/UInt8 "data: 5"

# 查詢停車任務進度
rostopic echo /mission/parking
```

### 🎮 8. control_moving（定速移動）
```bash
roslaunch control control_moving.launch
```

**訂閱:** `/control/moving/state` (MovingParam), `/odom`
**發佈:** `/cmd_vel`, `/control/max_vel`, `/control/moving/complete`
**MovingParam:** `moving_type` (0=Idle, 1=Left, 2=Right, 3=Forward, 4=Backward) + `moving_value_linear/angular`

**手動測試:**
```bash
# 前進 0.5m
rostopic pub /control/moving/state turtlebot3_autorace_msgs/MovingParam "moving_type: 3" "moving_value_angular: 0.0" "moving_value_linear: 0.5"

# 左轉 90度
rostopic pub /control/moving/state turtlebot3_autorace_msgs/MovingParam "moving_type: 1" "moving_value_angular: 90" "moving_value_linear: 0.0"
```

## 完整流程測試

### 方式一：core_node_controller（自動狀態機）
```bash
# 啟動所有基底
roslaunch turtlebot3_autorace_core turtlebot3_autorace_core.launch

# 啟動控制器
rosrun core core_node_controller.py

# 手動觸發第一關
rostopic pub /mission std_msgs/UInt8 "data: 1"
```

### 方式二：手動逐關測試
```bash
# 關1: 紅綠燈
roslaunch detect detect_traffic_light.launch
# → 等待 8 秒超時或收到綠燈

# 關2: S曲線
roslaunch detect detect_lane.launch
roslaunch detect detect_intersection.launch
# → 等待 30 秒超時或偵測到出口

# 關3: 施工區
roslaunch detect detect_construction.launch
# → 等待 30 秒超時

# 關4: 停車場
roslaunch detect detect_parking.launch
roslaunch control control_moving.launch
# → 等待 30 秒超時

# 關5: M曲線
roslaunch detect detect_lane.launch
roslaunch control control_lane.launch
# → 等待 30 秒超時

# 關6: 平交道
roslaunch detect detect_level.launch
# → 等待 30 秒超時

# 關7: 隧道
roslaunch detect detect_tunnel.launch
# → 等待 30 秒超時

# 關8: 終點
rostopic pub /mission std_msgs/UInt8 "data: 8"
```

## 快速診斷指令

```bash
# 看所有 running nodes
rosnode list

# 看所有 topics
rostopic list

# 看 /mission 最後消息
rostopic echo /mission --once

# 看 /control/lane 偏移量
rostopic echo /control/lane

# 看 /cmd_vel 速度
rostopic echo /cmd_vel

# 監控多個 topics（每秒更新）
rostopic monitor /mission /control/lane /cmd_vel

# 查看特定節點的輸出
rosnode info /detect_traffic_light
```

## 節點 → Mission 值對照表

| Mission | Enum 值 | 觸發節點 | 預期行為 |
|---------|---------|---------|---------|
| Idle | 0 | - | 等待 |
| TrafficLight | 1 | detect_traffic_light | 綠燈亮 → 發 Mission 2 |
| S_Curve | 2 | detect_intersection | 看到出口 → 發 Mission 3 |
| Construction | 3 | detect_construction | 障礙通過 → 發 Mission 4 |
| Parking | 4 | detect_parking | 停車完成 → 發 Mission 5 |
| M_Curve | 5 | detect_parking / 超時 | 30秒後自動進下一關 |
| LevelCrossing | 6 | detect_level | 柵欄升起 → 發 Mission 7 |
| Tunnel | 7 | detect_tunnel | 隧道出口 → 發 Mission 8 |
| Finish | 8 | detect_tunnel / 超時 | 停車，關閉所有節點 |

## 各節點依賴關係

```
camera (/camera/image/compressed)
    │
    ├──► detect_traffic_light  ──► /mission, /cmd_vel
    ├──► detect_lane            ──► /control/lane
    ├──► detect_intersection    ──► /mission, /control/lane, /cmd_vel
    ├──► detect_construction    ──► /mission, /control/lane
    ├──► detect_parking        ──► /mission, /cmd_vel, /control/moving/state
    ├──► detect_level          ──► /mission, /control/max_vel
    └──► detect_tunnel          ──► /mission, /control/cmd_vel

/odom ────────────────────────────────────────────────────►
    └──► control_moving          ──► /cmd_vel

/control/lane ─────────────────────────────────────────────►
    └──► control_lane           ──► /cmd_vel

/mission ─────────────────────────────────────────────────►
    └──► core_node_controller   ──► 啟閉各 detect_* 節點
```
