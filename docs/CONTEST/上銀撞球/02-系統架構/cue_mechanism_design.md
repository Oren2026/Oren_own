## v2.0 核心變更：力道可控 + 參數回饋

### 為何要升級力道控制

純手臂撞擊（v1.0）的問題：
- 力道取決於手臂速度 → 低速時位置精度差，力道解析度粗糙
- 每次撞擊的一致性依賴手臂重複精度
- 無法對「彈簧蓄能係數」「摩擦係數」做物理建模

v2.0 的核心思路：**把力道控制從手臂抽出來，交给彈簧機構**。

```
手臂職責：移動到定位、姿態控制
力道職責：彈簧蓄能機構（力道與手臂速度脫鉤）
回饋職責：力道感測器 + 視覺回授 → 建立物理模型
```

## 1. 機構概覽（v2.0 力道可控版）

### 1.1 設計目標

設計一套安装於 HIWIN 智慧機器手臂末端法蘭的撞球竿機構，實現：
- **力道可控**：撞擊力道由彈簧預壓量決定，與手臂速度脫鉤
- **參數回饋**：力道感測器 + 視覺觀測，可用於物理模型擬合
- **撞球竿穩定夾持與快速更換**

## 2. 與 HIWIN 手臂的連結方式

### 2.1 HIWIN 法蘭規格（推估）

根據上銀智慧機器手臂系列（如 HH100 等機型），末端法蘭典型規格：

| 參數 | 數值（推估） | 備註 |
|------|-------------|------|
| 法蘭直徑 | 50 mm | 標準 ISO 9409-1-50-4-M6 |
| 安裝孔 PCD | 40 mm | 4 孔 M6 |
| 螺絲規格 | M6 × 4 孔 | 螺絲深度建議 ≥ 10mm |
| 法蘭厚度 | 20 mm | 需確認 |
| 最大允許扭矩 | 待確認 | 需查表 |

> **重要提醒**：上述為典型工業機械手臂法蘭規格，**實際規格需以主辦單位提供的 HIWIN 手臂正式文件為準**。建議取得手臂 Docs 之後量測法蘭實際尺寸。

### 2.2 法蘭轉接板設計

**設計思路**：製作一個轉接法蘭，一側匹配 HIWIN 法蘭，另一側提供本機構的安裝介面。

```
轉接板俯視圖：
        ○ ← HIWIN 安裝孔（PCD=40mm, 4×M6）
      ╱  ╲
     ○────○  ← 本機構安裝孔（建議 M5 螺絲，4孔）
      ╲  ╱
        ○

轉接板側視圖：
    ┌─────────────┐
    │             │ ← 高度建議 15-20mm（提供機構空間）
    │   中心通孔   │ ← 可選，用於走線或定位銷
    │             │
    └──────┬──────┘
           ↓
      HIWIN 法蘭面
```

**轉接板材料選項**：

| 材料 | 優點 | 缺點 | 建議 |
|------|------|------|------|
| 6061 鋁合金 | 高強度，散熱好，精度高 | 需 CNC 加工，時間/成本較高 | 優先考慮（安全係數高） |
| PETG (3D列印) | 韌性好，耐衝擊，適合機構件 | 強度較鋁低，長期負載可能變形 | 作為前期原型驗證 |
| PLA (3D列印) | 便宜，快速成型 | 脆，高溫可能軟化 | 不建議用於此零件 |

**安裝方式**：使用 HIWIN 法蘭原本的 4 顆 M6 螺絲直接鎖附轉接板。

### 2.3 夾持器與法蘭的連結

夾持器透過以下方式與轉接板連結：

1. **螺絲鎖附**（主要）：M5 螺絲 × 4 顆，穿過轉接板螺孔與夾持器基座鎖合
2. **定位銷**（輔助）：直徑 3mm 定位銷 × 2 顆，防止剪切力導致錯位
3. **快拆卡扣**（可選）：設計為可快速拆卸結構，方便賽前/赛中更換

## 3. 桿身材質與尺寸建議

### 3.1 撞球竿桿身設計

**幾何尺寸**：

| 參數 | 數值 | 說明 |
|------|------|------|
| 總長度 | 350 mm | 考量機械手臂可達範圍與室內撞球桌高度 |
| 外徑 | 12 mm | 標準撞球桿比例，握持部通常 25-30mm |
| 壁厚 | 1.5 mm | 輕量化與強度平衡 |
| 材質 | 碳纖維管（CF tube） | 高剛性、低重量、衝擊韌性好 |
| 內徑 | 9 mm | 配合桿頭連接部 |

**為何選擇碳纖維管**：
- 楊氏模量約 70-150 GPa，遠高於鋁合金（69 GPa）
- 密度約 1.6 g/cm³，比鋁（2.7 g/cm³）輕
- 振動衰減性好，撞擊後殘震小
- 可直接購買標準品（如 12mm OD × 350mm 長）

**替代方案**：鋁合金管（6061-T6），成本低但較重，振動衰減差。

### 3.2 桿頭（Tip）設計

桿頭是直接撞擊母球的部分，需要：

| 參數 | 數值 | 說明 |
|------|------|------|
| 直徑 | 10 mm | 標準撞球桿桿頭直徑約 9-12mm |
| 長度 | 15-20 mm | 提供足夠撞擊面積 |
| 材質 | 皮革或軟質樹酯 | 標準撞球桿頭材料 |
| 連接方式 | 螺紋 or 膠合 | 建議 M6 螺紋 + LOCTITE 膠 |

**市售替代**：可直接購買標準撞球桿頭（10mm 直徑），鑽孔攻 M6 螺紋後與碳纖維管連接。

### 3.3 桿身與夾持器的介面

```
碳纖維管端部放大圖：

    ┌─────────────────┐
    │  碳纖維管(CF)    │  ← 外徑 12mm, 壁厚 1.5mm
    │  ┌───────────┐  │
    │  │  鋁套筒    │  │  ← 內徑 9mm, 外徑 14mm, 長 30mm
    │  │  (6061)   │  │     陽極處理鋁
    │  └───────────┘  │
    └───────┬─────────┘
            │ M6 螺紋孔
            ↓
      夾持器夾爪區
```

鋁套筒用意：
1. 保護碳纖維管端部（應力集中區）
2. 提供可靠的螺紋連接（碳纖維無法直接攻牙）
3. 增加與夾持器的接觸面積

## 4. 撞擊機構（施力與直線撞擊）

### 4.1 撞擊原理

撞球竿的撞擊本質是**直線加速運動**：

```
撞擊序列：
1. 機械手臂移動撞球竿，使桿頭靠近母球（接觸前 5-10mm）
2. 機械手臂沿擊球方向做直線加速（加速度 a）
3. 達到目標速度 v 後，桿頭接觸母球，瞬間動量 transfer
4. 撞擊完成後，手臂減速並撤回
```

### 4.2 力量控制策略

| 控制參數 | 影響 | 實現方式 |
|----------|------|----------|
| 撞擊速度 v | 球前進距離（速度越快=力道越大） | 手臂末端速度設定 |
| 加速曲線 | 撞擊順暢度、撞擊一致性 | S-curve 或梯形速度規劃 |
| 撞擊角度 | 球的行進方向 | 機械手臂姿態控制 |
| 撞擊位置 | 球的旋轉（若有） | 視覺系統定位 |

**關鍵點**：撞球時**不需要額外線性致動器**，純粹利用機械手臂的運動能力執行撞擊。

### 4.3 直線撞擊實現方式

**方式一：純手臂運動（推薦）**

```
手臂姿態控制：
- 將撞球竿姿態鎖定（鎖住姿態，僅移動位置）
- 使用直線軌跡模式（move_lin）而非關節插補
- 在接觸前 1-2mm 停止，讓手臂慣性完成撞擊
```

**方式二：彈簧蓄能撞擊（備選）**

適用於需要更大撞擊力或更精確力道控制的場景：

```
彈簧蓄能機構示意：

    夾持器
       │
    ┌──┴──┐
    │撞球竿│
    └──┬──┘
       │ ← 彈簧（預壓縮）
    ┌──┴──┐
    │撞球桿│ ← 可來回滑動的撞擊頭
    └──┬──┘
       │ ← 撞擊球
       
機構說明：
- 撞擊前：彈簧預壓縮（手臂拉回撞球桿）
- 撞擊時：釋放，彈簧推動撞擊頭直線撞向球
- 優勢：力道可控，撞擊一致性好
- 劣勢：機構複雜，重量增加
```

**方式三：線性致動器（不推薦）**

增加獨立的線性致動器（如舵機+絲桿）用於撞擊動作。**此方式會增加重量與機構複雜度，不建議優先採用。**

### 4.4 撞擊時序建議

```
t0: 手臂移動至準備位置（桿頭距球 10mm）
t1: 手臂開始向擊球方向加速
t2: 桿頭接觸球，動量轉移
t3: 手臂繼續移動（但不影響球）
t4: 手臂減速並撤回
```

**重要**：接觸時手臂應處於**剛性狀態**（高剛性模式），避免手臂柔順模式導致撞擊能量被吸收。

## 6. 彈簧蓄能撞擊機構（v2.0 核心）

### 6.1 核心原理

```
拉桿後退（蓄力）→ 鬆開 → 彈簧推動撞球桿向前撞擊 → 撞球桿反彈復位
```

撞擊力道完全由**彈簧預壓量**決定，跟手臂速度無關。

好處：
- 每次撞擊後自動復位到原點，一致性極高
- 力道解析度由「拉桿距離」控制，精度可達 0.1mm
- 實驗室調參後，可在競賽現場快速切換力道參數

### 6.2 機構細部設計

```
撞球桿滑軌系統（側視圖）：

    ┌──────────────────────────────────────┐
    │  滑軌外殼（3D 列印 PETG）             │
    │  ┌────────────────────────────┐     │
    │  │                            │◄─┼── 撞球桿（滑動）
    │  │    ┌──┐              ┌──┐  │     │
    │  │    │彈│ ←───────────│撞│  │     │  ◄─ 彈簧（壓縮狀態）
    │  │    │簧│              │球│  │     │
    │  │    │室│              │桿│  │     │  ◄─ 撞球桿（撞擊端）
    │  │    └──┘              └──┘  │     │
    │  │                            │     │
    │  │  ●──●──●──●──●──●──●──●  │     │  ◄─ 線性滑軌導軸
    │  │  ○                        │     │  ◄─ 拉桿（由小型伺服控制）
    │  └────────────────────────────┘     │
    │                ↑                    │
    │            法蘭轉接板                 │
    └──────────────────────────────────────┘

撞球桿（俯視圖）：

    ┌─────────┬────────────────────────────────┐
    │         │                                │
    │ 撞球桿  │         碳纖維管（12mm OD）       │  ← 可在滑軌內滑動
    │ 夾持部  │                                │
    │         │                          ◄─┘  ◄─── 桿頭（撞球端）
    └─────────┴────────────────────────────────┘
```

### 6.3 力道控制機制

| 參數 | 控制方式 | 說明 |
|------|---------|------|
| 預壓量（拉桿距離） | 小型伺服馬達（舵機）拉動撞球桿 | 拉 1mm = 撞出去力道 E，比例關係由實驗決定 |
| 撞擊速度 | 彈簧彈力決定，非手臂速度 | 可實驗測得：預壓 5mm → 球速 X m/s |
| spin（側旋） | 較難，v2 暫不實現 | 未來可在撞球桿上加微型馬達做側向振動 |

**力道標定流程**：
1. 固定拉桿距離（例：3mm、5mm、7mm）
2. 各打 10 球，記錄球速（視覺測量）
3. 建立「拉桿距離 → 球速」對照表
4. 比賽時只要知道「要打多快」，查表得到「拉多少」

### 6.4 力道感測器（可選，回饋用）

在撞球桿尾端（不接觸球的那端）加裝：
- **應變規（Strain Gauge）**：測量撞擊瞬間的衝擊力
- **微型壓力感測器**：測量彈簧實際釋放力量

好處：不用視覺也能知道撞擊力道是否如預期，可用於封閉迴圈控制。

## 7. 物理模型擬合（比賽時即時參數識別）

### 7.1 為何要擬合參數

每張桌子的摩擦係數 μ 和庫邊反彈係數 e 不同。機械校正時在 A 桌打得準，不代表在 B 桌也準。

### 7.2 擬合流程

```
Step 1: 機器人自動打 5 球（固定力道，直線打向庫邊）
Step 2: 視覺系統觀測每次反彈後的方向和距離
Step 3: 用最小平方法（Least Squares）逆推這張桌子的 μ 和 e
Step 4: 以後每次計算軌跡時，使用識別出來的 μ 和 e
```

數學模型（高中物理就夠）：
```
球的運動衰減：v_new = e * v_old（庫邊碰撞）
摩擦減速：v_after = v_before * (1 - μ * d)（d = 行進距離）
```

### 7.3 軌跡規劃

拿到 μ 和 e 以後，每次打目標球前：
1. 計算理想撞擊角度（直射或折射）
2. 計算需要的球速（力道）
3. 查力道標定表得到「拉桿距離」
4. 執行撞擊

## 8. 風險點與替代方案

### 8.1 風險分析

| 風險項目 | 發生機率 | 影響程度 | 緩解策略 |
|----------|----------|----------|----------|
| **HIWIN 法蘭規格與預期不符** | 中 | 高 | 取得主辦單位文件後立即確認；設計時預留尺寸調整餘裕（槽孔） |
| **彈簧蓄能力道不足** | 低 | 中 | 更換更強彈簧；增加預壓量上限 |
| **撞球桿在滑軌上卡住** | 中 | 高 | 滑軌加潤滑；初期用 PTFE 耐磨片 |
| **撞球桿復位不精確** | 中 | 高 | 每次撞擊後撞球桿自動復位到零點（由舵機拉回） |
| **力道感測器雜訊過大** | 低 | 中 | 軟體濾波（移動平均或卡爾曼濾波） |
| **參數擬合收斂失敗** | 低 | 中 | 從多球取平均降低雜訊；設定擬合初值 |
| **桿頭磨損** | 高 | 低 | 多備幾個桿頭（3-5個）；使用可快速更換設計 |
| **3D 列印件強度不足** | 中 | 中 | 提升填充率；改用 PETG；必要時改用鋁合金 |

### 6.2 替代方案

| 方案 | 適用情境 | 優點 | 缺點 |
|------|----------|------|------|
| **純機械手臂撞擊（原方案）** | 多數情況 | 機構簡單，重量輕 | 力道依賴手臂性能 |
| **彈簧蓄能撞擊** | 需要更精確力道控制 | 力道可控性高，一致性好 | 機構複雜，重量增加 |
| **吸盤式（不推薦）** | 替代方案 | N/A | 球是光滑的，吸盤無效 |
| **直接用爪子夾球（不推薦）** | 替代方案 | N/A | 犯規，直接觸碰球是違規 |

### 6.3 設計餘裕建議

1. **法蘭轉接板**：安裝孔使用**橢圓形槽孔**（長槽），提供 ±2mm 調整空間
2. **夾持器**：預留更多夾持位置（可調整夾持高度）
3. **撞球竿長度**：設計為**可拆卸式**，可更換不同長度（300-400mm 範圍）
4. **桿頭**：設計為**快拆式**，30 秒內可更換

## 9. 下一步行動

### 短期（來得及 5/22 複審影片）

1. **確認 HIWIN 法蘭尺寸**（最高優先）：聯繫主辦單位或安全講習時實測
2. **製作法蘭轉接板**：鋁合金 CNC 或 3D 列印 PETG（先做第一版）
3. **驗證撞球桿滑動機構**：3D 列印測試，確認撞球桿來回滑動不卡
4. **簡化版撞擊測試**（純手臂速度）：先確認能在影片裡看見成功撞擊

### 中期（決賽前）

1. **整合彈簧蓄能機構**：選擇彈簧規格（Spring Rate k 和自由長度）
2. **力道標定實驗**：建立拉桿距離 → 球速對照表
3. **物理參數擬合測試**：在不同桌面上驗證 μ 和 e 的識別效果
4. **力道感測器整合**（可選）：閉環控制撞擊力道
