## RA605-GC 安裝法蘭規格

| 參數 | 數值 |
|------|------|
| 中心孔 | Ø20 mm（H7，公差）深度 4mm |
| 定位銷孔 | Ø5 mm（H7），深度 7mm，位於 12 點鐘方向 |
| 外部凸緣 | Ø40 mm（h8 公差），深度 6mm |
| 安裝螺孔 | 4 × M5，螺距 0.8P |
| 螺紋深度 | 6mm（有效） |
| PCD（螺孔分布圓） | Ø31.5 mm |
| 螺孔角度配置 | 45°（相對於中心軸線偏轉） |

## 設計轉接板注意事項

### 3D 列印 PETG 版

- **轉接板厚度**：8-10mm（螺絲有效深度 6mm，需留餘裕）
- **穿孔直徑**：Ø5.2~5.3mm（M5 螺絲外徑 5mm，PETG 韌性好）
- **攻牙**：PETG 可直接攻 M5 螺紋，或使用尼龍嵌入式螺帽
- **定位銷**：建議轉接板也做 Ø5 定位銷孔（12點鐘方向），確保機構不會轉動

### 機構干涉檢查

- PCD 31.5mm，轉接板外徑建議 38-45mm（不能太大，會跟手臂衝突）
- 中心孔 Ø20mm，轉接板中間需穿透

### 強度提醒

- 四孔 45° 分布對稱，抗扭矩能力足夠
- 撞球撞擊時有反作用力，PETG 厚度不足可能斷裂
- 建議轉接板加肋板（rib）強化結構
