# 撞球竿機構研究報告

## 研究目標
- 搜尋現有撞球機器人的撞球竿機構設計
- 找出 HIWIN 手臂的 API 或控制方式
- 整理設計要點

## 一、現有撞球機器人機構案例

### 1.1 GitHub 開源專案

#### 專案 1: `opticsensors/pool-playing-robot`
- **架構**: Python (視覺) + C++ (路徑規劃) + Arduino (馬達控制)
- **驅動方式**: 兩個步進馬達 + 電磁閥 (solenoid) 擊球
- **關鍵技術**:
  - Arduino 控制步進馬達 (STEP/DIR 介面)
  - 使用 AccelStepper 函式庫
  - 支援微步進 (microstepping)
  - 電磁閥控制桿球撞擊力道與時序
- **CAD 檔案**: 提供 SolidWorks 模型與 3D 列印零件

#### 專案 2: `kevinwang408/Robotic-Arm-Billiards-Competition`
- **系統組成**:
  1. 影像處理 (Python): 偵測球與桌檯位置
  2. 路徑規劃 (C++): 計算最佳擊球軌跡
  3. 機器手臂控制 (C++): 執行擊球動作
- **擊球策略**:
  - 直線射擊 (Direct Shot)
  - 折射射擊 (Reflection Shot) - 考慮牆面反彈
  - 障礙物檢測演算法

#### 專案 3: `TenYearsADream/billiard_robot`
- 使用機械手臂打撞球
- 提供機構示意圖

#### 專案 4: `gwilly247rm/Robot_arm_pool_game`
- 專案名稱: "hiwin_pool_game"
- 直接使用 HIWIN 手臂

## 二、常見撞球竿機構方案

### 2.1 機構類型

| 類型 | 說明 | 優點 | 缺點 |
|------|------|------|------|
| 夾持式 (Gripper) | 機構末段夾持撞球桿 | 置換方便 | 剛性較低 |
| 剛性連接式 | 撞球桿直接固定於手臂末端法蘭 | 剛性高、精度好 | 需要精確對齊 |
| 滑軌推拉式 | 撞球桿在滑軌上直線運動擊球 | 穩定可靠 | 需要額外致動器 |
| 複合式 | 結合夾持與線性致動 | 靈活性高 | 機構複雜 |

### 2.2 桿身材質建議

| 材質 | 特性 | 適用性 |
|------|------|--------|
| 碳纖維管 | 輕量、高剛性、低熱膨脹 | 優先推薦 |
| 鋁合金管 | 輕量、易加工 | 可用 |
| 鈦合金管 | 高剛性、高強度 | 成本較高 |
| 3D 列印 PLA/ABS | 快速原型製作 | 僅用於初期測試 |

### 2.3 驅動方式

1. **手臂本體驅動**: 利用機械手臂 6 軸運動產生擊球力道
   - 需控制撞擊速度與角度
   - 難度較高

2. **額外致動器驅動**:
   - **步進馬達**: 直線推拉撞球桿
   - **伺服馬達**: 更精確的速度控制
   - **氣壓/液壓致動器**: 爆發力強
   - **電磁閥 (Solenoid)**: 快速撞擊

## 三、撞球桿機構設計要點

### 3.1 幾何關係
- **撞擊點計算**: 根據目標球位置與口袋位置，計算桿球撞擊點
- **切角 (Cut Angle)**: 影響擊球精度
- **桿球角度**: 撞球桿與桿球的角度關係

### 3.2 機構精度要求
根據 `opticsensors/pool-playing-robot` 的誤差分析:
- **視覺系統解析度**: ≤ 0.5 mm
- **致動系統解析度**: ≤ 0.25 度
- **切角影響**: > 60° 時誤差急遽增加

### 3.3 擊球動作時序
```
1. 移動手臂至撞擊位置
2. 撞球桿後拉 (蓄力)
3. 撞球桿前推撞擊桿球
4. 擊球後撞球桿收回
```

## 四、HIWIN 手臂控制

### 4.1 控制方式
HIWIN 工業手臂通常支援以下控制介面:
- **TCP/IP**: 乙太網路控制
- **Modbus TCP**: 工業通訊協議
- **SDK/API**: 原廠提供之軟體開發套件
- **GPIO**: 數位輸入輸出控制

### 4.2 上銀智慧機器手競賽
- 第十八屆上銀智慧機器手實作競賽 撞球組
- 主辦單位提供 HIWIN 智慧機器手臂
- 截止: 2026-05-22 複審影片，2026-08-19~20 決賽

### 4.3 控制策略建議
1. 使用手臂末端的 Tool Center Point (TCP) 控制撞球桿姿態
2. 利用線性軌跡規劃確保擊球穩定性
3. 建議結合外部感測器做視覺回授

## 五、推薦機構方案

### 5.1 方案 A: 剛性連接 + 手臂本體驅動
- 撞球桿直接固定於手臂法蘭
- 由手臂 6 軸運動產生擊球力道
- 適用於精度較高之應用

### 5.2 方案 B: 夾持式 + 額外致動器
- 機構末段夾持撞球桿
- 撞球桿可在夾持機構中滑動
- 由小型致動器控制撞擊

### 5.3 方案 C: 滑軌複合式 (推薦)
- 撞球桿固定於線性滑軌
- 由步進馬達或伺服馬達推拉
- 可獨立控制撞擊力道與速度
- 機構如 `opticsensors/pool-playing-robot` 所示
