# i18n — 國際化框架

## 設計原則

所有使用者看到的 UI 字串，都從 `locales/` 目錄下的 JSON 讀取，不在 code 裡 hardcode。

---

## 目錄結構

```
frontend/src/locales/
├── zh-TW.json    繁體中文（預設）
├── zh-CN.json    簡體中文
├── en.json       英文
├── ja.json       日文
└── vi.json       越南文
```

## 前端使用方式

```javascript
// 簡單替換
const t = window.i18n.get;
element.textContent = t('common.submit');

// 含變量
const msg = t('time.minutes_ago').replace('{n}', 5);
// → "5 分鐘前"
```

## 學校預設語言

```sql
users.locale       -- 使用者個人偏好（可覆寫 school 預設）
schools.locale     -- 學校預設 UI 語言
```

登入時：`users.locale ?? schools.locale ?? 'zh-TW'`

---

## 時區處理

```sql
schools.timezone    -- IANA format：'Asia/Taipei' / 'America/New_York'
users.timezone      -- 個人時區偏好
```

所有時間在 DB 裡以 **UTC 儲存**。\
展示時轉換：
```javascript
const tz = user.timezone || school.timezone || 'UTC';
const localTime = new Date(utcString).toLocaleString('zh-TW', { timeZone: tz });
```
