## 🤖 任務關卡

| 任務 | 說明 |
|------|------|
| 車道置中 | 維持車子在車道中央行駛 |
| 紅綠燈辨識 | 辨識交通號誌並做出反應 |
| 左右轉辨識 | 辨識轉向標誌並執行轉彎 |
| 路障辨識 | 偵測障礙物並繞行 |
| 山洞穿梭 | 通過狹窄或暗區域 |
| 停車 | 到達指定位置停車 |

## ⚙️ 系統架構

### 狀態機任務系統

```
任務管理器 (task_manager.py)
  └── 任務清單 (mission_list.py)
        ├── Step 1: 車道置中
        │     → 條件達成 → 關閉 + 開啟下一步
        ├── Step 2: 紅綠燈辨識
        │     → 條件達成 → 關閉 + 開啟下一步
        ├── Step 3: 左右轉辨識
        │     → 條件達成 → 關閉 + 開啟下一步
        ├── Step 4: 路障辨識
        │     → 條件達成 → 關閉 + 開啟下一步
        ├── Step 5: 山洞穿梭
        │     → 條件達成 → 關閉 + 開啟下一步
        └── Step 6: 停車
              → 完成
```
