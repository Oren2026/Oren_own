🚦 detect_traffic_light 實作細節

方法 功能
fnMaskRed/Yellow/GreenTrafficLight HSV 顏色空間過濾，產生二值化遮罩圖
fnFindCircleOfTrafficLight 霍夫圓變換（Hough Circle Transform）確認真實燈號
fnFindTrafficLight 調度三色偵測，決定目前燈號
cbGetImage 接收 /image_input，轉 OpenCV 格式
cbGetDetectTrafficLightParam dynamic_reconfigure 動態調整 HSV 閾值

輸出 Topic：

Topic類型說明
/cmd_velTwist直接控車（綠燈時往前）
/missionUInt8任務切換（完成後發下一個 Mission ID）
/detect/traffic_light/compressedCompressedImage視覺化偵測結果

🎮 core_node_controller 實作細節

每個 Mission 完成後，該節點用 roslaunch 啟閉下一個偵測節點：


核心邏輯： Mission = Enum('Mission', 'Idle TrafficLight Intersection Construction Parking LevelCrossing Tunnel') cbReceiveMode(mode_msg) → 設定 current_mode → is_triggered = True fnControlNode() → 根據 current_mode 判斷要 launch 哪個節點 → 呼叫 fnLaunch(launch_num, is_start) fnLaunch(launch_num, is_start) → roslaunch.scriptapi.ROSLaunch() → roslaunch.parent.ROSLaunchParent(uuid, [path]) → .start() 或 .shutdown()
