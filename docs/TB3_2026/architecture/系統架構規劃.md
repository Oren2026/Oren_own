## ⚙️ 軟體架構

### 狀態機任務系統

```
┌─────────────────────────────────────────────────────────┐
│                    任務管理器                              │
│                   task_manager.py                         │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                    任務清單                               │
│                  mission_list.py                          │
│  ┌─────────────────────────────────────────────────┐     │
│  │ class MissionState:                              │     │
│  │   - name: 任務名稱                               │     │
│  │   - active: 是否啟動                             │     │
│  │   - conditions: 觸發條件列表                      │     │
│  │   - check_complete(): 檢查是否完成               │     │
│  │   - activate(): 啟動此任務                       │     │
│  │   - deactivate(): 關閉此任務                     │     │
│  └─────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────┘
```

### 模組結構

```
TB3_AutoRace/
├── task_manager.py        ← 主管理器
├── mission_list.py        ← 任務清單（狀態機核心）
│
├── vision/                ← 視覺模組
│   ├── lane_detection.py  ← 車道線偵測
│   ├── traffic_light.py    ← 紅綠燈偵測
│   ├── turn_sign.py       ← 左右轉標誌偵測
│   ├── obstacle.py         ← 路障偵測
│   ├── tunnel.py           ← 山洞偵測
│   └── parking.py          ← 停車位置偵測
│
├── control/               ← 控制模組
│   ├── lane_following.py  ← 車道居中控制
│   ├── turning.py          ← 轉向控制
│   ├── obstacle_avoid.py  ← 繞行控制
│   └── parking_control.py ← 停車控制
│
└── sensors/               ← 感測器模組
    ├── lidar.py           ← 雷射雷達
    ├── imu.py             ← IMU 姿態
    └── camera.py          ← 相機整合
```

## 🔧 技術選擇

### 視覺處理流程

```
Camera (rqt 調整參數)
    │
    ▼
OpenCV 圖像處理
    │
    ├── 顏色空間轉換 (BGR → HSV)
    ├── 邊緣偵測 (Canny)
    ├── 輪廓提取 (findContours)
    └── 物件辨識 (模板匹配 / CNN)
    │
    ▼
辨識結果
    │
    ▼
任務條件判定
    │
    ▼
狀態機切換
```

### rqt 圖像調整

- 使用 `rqt_image_view` 調整相機參數
- 曝光、白平衡、解析度
- 確認視覺閾值設定
