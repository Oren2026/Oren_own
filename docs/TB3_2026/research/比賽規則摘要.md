# 比賽規則摘要

## 賽道圖

終點線位於上方紅色實線。行進方向：**逆時針**。

（圖片：`assets/images/track-map.jpg`）

## 逆時針行進流程

依逆時針方向，機器人從上方紅線起點出發，依次經過：

1. 🚦 **交通號誌** — 起點左側三色號誌燈，紅燈停、綠燈行
2. 🚧 **障礙區** — 左側三個紅色三角障礙標誌
3. 〰️ **S 型彎道** — 左側連續彎道，跟隨黃白車道線行進
4. 🅿️ **停車區** — 底部岔路，白色虛線車格，藍色 P 標誌入口，定點停入
5. ↩️ **圓環/路口** — 藍色圓形箭頭標誌，識別方向並決策出口
6. 🚞 **平交道** — 右下角紅白相間柵欄，識別並等待通行
7. 〰️ **M 型急彎** — 右下角鋸齒狀 M 型區域，狹窄彎道控制
8. 🚇 **隧道** — 右上方大面積黑色區域，光照補償失效時需備援機制
9. 🏁 **返回終點線** — 繞完一圈回到上方紅線

> ⚠️ 賽道中央左側有一個內部迴圈（禁制標誌＋方向箭頭），為高階關卡，基礎版本先繞外圈即可。

## 地面標線

| 顏色 | 位置 | 意義 |
|------|------|------|
| 黃線 | 行進方向左側 | 車道左邊界 |
| 白線 | 行進方向右側 | 車道右邊界 |
| 紅線 | 起點/終點（上方） | 起終點線 |
| 青藍色條 | 左側垂直路段 | 減速帶/斑馬線 |

## 核心技術

| 技術 | 用途 |
|------|------|
| HSV 顏色空間 | 紅/黃/綠燈號偵測，比 BGR 更穩定 |
| 霍夫圓變換 | 在顏色遮罩中找圓形，確認是否為真實燈號 |
| 透視轉換 | 將視角轉為 top-down，擠出消失點 |
| 滑動視窗 | 找出左側黃線、右側白線，計算中心線 |
| PID 控制 | 偏離中心線輸出 steering angle |
| 光源補償 | 隧道區曝光補償，須有備援機制 |
| 障礙物偵測 | 紅色三角標誌辨識 |
