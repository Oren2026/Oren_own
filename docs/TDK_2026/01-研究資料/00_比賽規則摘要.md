## 比賽主題

**讓世界看見雲林**

## 組別

| 組別 | 說明 |
|------|------|
| 自動組 | 機器人全自動執行任務 |
| 遙控組 | 機器人由選手遙控操作 |

## 任務關卡（自動組）

根據規則文檔，存在以下關卡：

1. **文蛤分級關卡**
   - 文蛤以 3D 列印製作（厚度3.5cm灰色、厚度1.5cm黑色）
   - 機器人需將文蛤掃落至篩選區
   - 使用3cm間隙的篩網進行一階段過篩

2. **古坑咖啡雲科店**
   - 黑白咖啡盤擺放
   - 視覺圖卡辨識
   - 點餐單處理

3. **古坑柳丁園**
   - 柳丁樹相關任務
   - 須有摘取柳丁功能之機構（強制規定）

4. **西螺七崁群英會**
   - 包含髮夾彎、英雄台等障礙
   - 貴賓布偶攜帶任務

5. **西螺大橋**
   - 斜坡行走

## 約束條件

- 摘取柳丁功能之機構為強制規定
- 機器人須全程釋放物件才算完成任務
- 場地尺寸可能有些許誤差

## 評分重點

- 完成任務數量
- 創意獎（靜態評審）
- 機器人論文（獲獎隊伍需繳交4-8頁論文）
