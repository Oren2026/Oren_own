## 🎯 四大關卡與所需機構

### 關卡 1｜成龍濕地生態守護
- **任務 A：** 扶正 3 個倒下的直立桿狀裝置藝術 → 弧形夾指
- **任務 B：** 將尖叫雞、天鵝娃娃送往救護站 → 托盤式底爪
- ⚠️ 機構需具備垂直升降能力

### 關卡 2｜文蛤分級
- **任務：** 將集中的文蛤掃入篩選區，透過不同孔徑篩網分級
- 所需機構：推板（大面積平板，非夾爪）

### 關卡 3｜稻草卷堆放
- **任務：** 將散落的稻草卷堆成三角形結構
- 所需機構：弧形夾爪（內側圓弧配合圓柱體）

### 關卡 4｜北港迎媽祖
- **任務 A：** 文旦、柳丁、砂糖橘放入供桌水果盤 → 水果夾持機構
- **任務 B：** 擲筊動作 → 純舵機雙爪開合控制
- ⚠️ 文旦較重，吸盤可能比夾爪穩定

## 🏗️ 整體架構（遙控組）

```
┌──────────────────────────────────────────────┐
│              🎮 遙控器（操作者）                  │
│         （接收影像 + 操作機器人移動）             │
├──────────────┬──────────────┬─────────────────────┤
│   影像回傳   │   移動底盤   │    機構控制模組      │
│  即時監控    │  DC馬達驅動  │  伺服馬達 × 多軸    │
│              │  履帶/輪子   │  爪手更換系統        │
└──────────────┴──────────────┴─────────────────────┘
        │
        ▼
┌──────────────────────────────────────────────┐
│              主控制系統（RPI 5）              │
│    （訊號中繼 + 機構動作控制 + 姿態監測）       │
├──────────────┬──────────────┬──────────────────┤
│   姿態感測   │   訊號處理   │   機構動作控制     │
│   IMU        │  遙控訊號解碼 │  舵機驅動輸出     │
└──────────────┴──────────────┴──────────────────┘
```

說明：操作者遙控移動，機構爪子由操作者或預程式控制。RPI 5 負責訊號路由、姿態監控、機構協調。
