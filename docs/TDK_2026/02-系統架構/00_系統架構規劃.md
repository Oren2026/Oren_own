## ⚠️ 重要聲明

**這不是 TurtleBot 3 比賽！**

TDK 盃是「讓世界看見雲林」主題的機器人競賽，主題與雲林農業相關（文蛤分級、柳丁採收等），**需要自行設計與製作機器人**，非使用市售的 TurtleBot 3 平台。

---

## 🎯 核心任務分析

根據官方公告，本屆 TDK 涉及以下任務：

| 任務 | 描述 | 需要的技術 |
|------|------|-----------|
| 文蛤分級 | 將不同尺寸/顏色的文蛤進行分級過篩 | 視覺辨識、物料分揀機構 |
| 柳丁摘取 | 從樹上摘取柳丁 | 機構設計、夾爪控制 |
| 西螺大橋 | 通過橋樑結構障礙 | 移動底盤、爬坡能力 |
| 咖啡店任務 | 放置咖啡盤、圖卡辨識 | 視覺辨識、精確定位 |
| 英雄台挑戰 | 攜帶貴賓布偶登上台階 | 穩定移動、機構設計 |

## 📋 預計系統架構

### 整體架構

```
┌─────────────────────────────────────────────┐
│              主控制系統                       │
│           （STM32 / ESP32）                  │
├─────────────┬─────────────┬───────────────────┤

```

## 🔧 預計技術需求

### 移動底盤
- 需要能通過橋樑斜坡
- 需要穩定的室內導航能力
- 建議：麥克林輪底盤 或 履帶底盤

### 視覺系統
- 深度相機（用於物體辨識和定位）
- 可能需要：Intel RealSense / Azure Kinect / Logitech  webcam

### 機構設計
- 夾爪機構（用於柳丁摘取、文蛤分級）
- 篩選機構（文蛤分級用）
- 承載平台（貴賓布偶）

### 控制系統
- 主控：STM32 或 ESP32
- 感測器整合：IMU、超音波、IMU

---

## 📐 預計需要的零件

### 核心
- 移動底盤（自製或改造）
- 主控制器（STM32/ESP32）
- 深度相機

### 驅動
- 直流馬達 × 4（輪子驅動）
- 伺服馬達 × 4-6（夾爪/機構）

### 感測器
- IMU 模組
- 超音波感測器 × 4
- encoder（馬達轉速回饋）

### 機構
- 3D 列印零件
- 金屬結構件
- 夾爪
