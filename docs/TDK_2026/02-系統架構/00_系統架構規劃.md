# TDK 2026 系統架構

> 狀態：規劃中
> 建立日期：2026-03-26

## 預計使用技術

| 項目 | 技術 |
|------|------|
| 硬體平台 | TurtleBot 3 |
| 控制系統 | ROS (Robot Operating System) |
| 視覺辨識 | OpenCV |
| 嵌入式 | Arduino / ESP32 |
| 機構 | 3D 列印元件 |

## 預計功能模組

### 1. 移動控制模組
- SLAM 定位
- 路徑規劃
- 障礙物檢測

### 2. 視覺辨識模組
- 物體辨識（文蛤、柳丁等）
- 顏色辨識
- 圖卡辨識

### 3. 機構控制模組
- 夾爪控制
- 篩選機構
- 柳丁摘取機構

### 4. 任務決策模組
- 任務狀態管理
- 決策邏輯
- 時間管理
