# TDK 2026 比賽規劃

> 第30屆 TDK 盃全國大專校院創思設計與製作競賽
> 主題：讓世界看見雲林

## ⚠️ 重要提醒

**這不是 TurtleBot 3 比賽！**

本屆 TDK 需要**自行設計與製作機器人**，主題與雲林農業相關（文蛤分級、柳丁採收等），非使用市售的 TurtleBot 3 平台。

## 主要關卡

1. **西螺大橋** — 通過橋樑結構障礙
2. **古坑咖啡雲科店** — 咖啡盤放置、圖卡辨識
3. **古坑柳丁園** — 柳丁摘取任務（強制要求）
4. **文蛤分級（自動組）** — 3D 列印文蛤分級過篩
5. **西螺七崁群英會** — 英雄台台階挑戰、貴賓布偶任務
