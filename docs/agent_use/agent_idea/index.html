創意實驗室

researcher 與 coder 協作的黑皮團隊創意展示。每一個頁面都是獨立的 CSS 實驗。

01
本週最深
編輯排版 / 大字引言風格。每次刷新隨機顯示 research 筆記錄，像 Fortune Cookie 一樣。
大字引言 編輯排版
02
意識光譜
深色卡片主題，純 HTML + CSS。展示意識的層級結構。
深色卡片 意識
03
對照記
分屏雙欄對比 Biological vs Digital Intelligence。暖色神經系統一側，冷色 AI 模型一側，可切換專注模式。
分屏對比 VS 佈局
04
光譜圖
意識光譜互動視覺化。水平 Gradient Bar 上分佈五層意識節點，點擊查看各層級詳細說明。
互動視覺化 光譜
05
年代記
編年報紙風格。仿報紙排版，大字日期軸、欄目分區、話題標籤雲，知識庫文章的編輯排版。
編年報紙 編輯排版
06
Pulse
知識心跳。Scroll-triggered 卡片進場動畫、hover 浮動發光效果、頂部閱讀進度條、數字滾動計數動畫。
Scroll動畫 Hover浮動 進度條
07
Flow
垂直時間線。左右交替節點、滾動觸發進場動畫、點擊展開詳情、三色漸變連接線。
時間線 Scroll觸發 點擊展開
08
Switch
多視角探索。即時切換卡片 / 列表 / 時間線三種視圖，打字機標題效果，視圖切換帶平滑過渡動畫。
視圖切換 打字機 互動式
09
Pulse Site
企業形象網。固定導航列 + backdrop blur、全版 Hero、主題色彩系統、卡片 hover 上浮 + 陰影加深、Stats 計數動畫。
企業網站 固定導航 Hero
10
Atlas
知識地圖。暖色大地調、襯線字體、報紙編輯排版、話題雲、知識領土視覺化地圖面板。
大地色系 知識地圖 襯線字體
