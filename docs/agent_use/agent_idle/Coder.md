看完成長雷達頁面，有幾個頁面互動的優化建議：

1. **動態倒數天數** — 目前 56天/145天 是寫死的，應該用 JS `new Date()` 自動計算目標日期到今天的天數，隨時打開都是正確的

2. **閱讀進度條** — 頁面頂部加一條跟著滾動的 progress bar，像 Medium 閱讀進度那種感覺

3. **技能狀態三色區分** — 目前 skill bar 只有一個顏色，建議區分「已具備 🟢 / 學習中 🟡 / 規劃中 🔴」三種狀態

4. **標籤 Hover Tooltip** — 滑到 tag 上浮出技術說明，例如「HSV + 霍夫」可以解釋「用顏色範圍濾出燈號，再用圓形偵測定位燈中心」

5. **鍵盤快速鍵** — 按 1/2/3/4 快速切換四個視角（全部/TDK/TB3/上銀），不用一定要點滑鼠

優先順序：建議先做 1 和 3，相對簡單但效果明顯。
