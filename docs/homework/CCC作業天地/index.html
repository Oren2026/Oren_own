使用說明

  • 老師的作業要求放在對應子資料夾，同學的繳交內容建議另外建立資料夾存放
  • 每次改版前建議先複製前版再做修改，保留完整演化軌跡
  • index.html 只放最新版本入口；各版本詳細說明在各版本資料夾的 README.md
  • 更新版本時：在 README.md 頂部的「改版日誌」表格加入新列
  • 如需上傳大型檔案，建議使用 Git LFS 或外部雲端連結
