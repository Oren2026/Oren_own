核心概念
一個在瀏覽器裡就能跑的日誌系統，從初版一路演化到 v3.0。前端用 JavaScript + sql.js（SQLite 的 WASM 版）實作，後期加入 Node.js 後端對照版。前端版本無需安裝任何東西，開了瀏覽器就能用。
純前端 SQLite (WASM) 本地持久化 亮暗主題 SPA 路由 Hash 模式
功能總覽
🌙

亮 / 暗版主題

一鍵切換亮版與暗版，CSS 變數驅動，無需重整頁面。

👥

作者追蹤系統

可追蹤 / 取消追蹤作者，有自己的個人頁（`/#/profile/:username`），顯示粉絲數、追蹤數、文章列表。

🔗

Hash SPA 路由

純前端路由，無需後端支援，URL 直接分享到特定頁面。

🔐

JWT 身份驗證

Node.js 後端版支援 JWT，bcrypt 密碼雜湊，完整的使用者系統。

🗄️

本地優先儲存

sql.js + IndexedDB，資料存在瀏覽器本地，關掉後端也能完整使用（試用模式）。

🎨

自定義個人頁

可自行設定頭像色彩、背景圖、暱稱與自我介紹。
