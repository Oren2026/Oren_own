核心概念
一個純前端的 JavaScript 測驗系統，20 題題庫每次隨機抽出 10 題，選項亂序排列。支援進度條跳題、即時動畫回饋、結算分數與重新測驗。純瀏覽器執行，無需後端。
純前端 隨機題目 亂序選項 即時回饋 進度追蹤
功能總覽
🎲

隨機題目與亂序選項

20 題題庫每次抽出 10 題，每題的四個選項順序也隨機打亂，降低背答案的可能。

📊

進度條與跳題

頂部進度條即時顯示目前進度，可隨時點擊題號跳回前一題修改答案。

⚡

即時動畫回饋

選中答案後立即顯示對錯，帶有平滑動畫效果，讓學習過程更有感。

🏆

結算分數

作答完畢後顯示總分、答對題數、正確率，以及所有題目的正解說明。

🔄

重新測驗

一鍵重新開始，題目與選項再次隨機，重新計時。
