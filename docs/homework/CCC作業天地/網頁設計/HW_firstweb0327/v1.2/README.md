# v1.2 — 網頁日誌系統

> 更新日期：2026-03-27｜從 v1.1 新增功能

---

## 📋 版本說明

從 v1.1 新增「作者過濾」功能，以及更新的預設文章內容。

---

## ✨ v1.2 新功能

| 功能 | 說明 |
|------|------|
| 👤 **作者過濾** | 點擊作者名稱 → 只看該作者文章，右上角 ✕ 清除 |
| 📝 預設文章更新 | 10篇文章，4位作者，內容與雲林相關 |

---

## ✨ 完整功能一覽

| 功能 | 說明 |
|------|------|
| 👤 作者過濾 | 點作者名稱只看該作者，右上角 ✕ 清除 |
| 🌍 公開日誌牆 | 所有人的文章都顯示 |
| 🔐 Auth Modal | 浮動雙面板登入/註冊 |
| ➕ 發文 | 登入後可發表、編輯、刪除自己的文章 |
| 🔍 搜尋 | 即時過濾標題與內容 |
| 💾 匯出/匯入 | SQLite 資料庫備份與還原 |
