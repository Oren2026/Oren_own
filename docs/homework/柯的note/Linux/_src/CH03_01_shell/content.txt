shell 下有些特殊字元,補系統指令之不足
#：標示註解, 這些說明不會被當成指令或程式執行
&：背景方式執行,例如執行 sudo updatedb & 指令
'：單引號定義引號內為一個完整的字串, 並且讓 shell 不要解讀其中的空白或特殊字元
*：對應任何字串、字元或空字串
?：對應任何單一字元
.：表示目前所在的目錄
..：表示上一層的目錄
;：可用它將多個指令分隔開來, 讓指令依序執行
>：把執行結果輸入一個檔案中
>>：把執行結果附加到一個檔案後面
\：脫逸符號 (escape), 解除特殊字元的含意
|：建立管線, 將前一個指令的執行結果輸入給下一個指令使用
~：使用者登入後所在的目錄, 也就是使用者家目錄
[英文字母或數字]：對應括弧中任意範圍的字元
也可以指定字母的範圍
也可以指定數字的範圍
[!英文字母或數字]：排除括弧中任意範圍的字元
{英文字母或數字}：對應括弧中的任意字元
