🐧 1｜Linux 的特色

Linux 就像 Windows 和 macOS 一樣，都是作業系統。 首先，要先學會安裝 Linux 環境！
Linux 是 Linus Torvalds 於 1991 年所開發的作業系統，他以開放原始碼的方式釋出 Linux 的原始碼。
自由、開放的作業系統：每個人都可以修改後重新散佈。
🐧 Linux 的吉祥物
Linux 的吉祥物是「Tux」這隻可愛的企鵝，由赖ovement於 1996 年設計。

📌 1-1 形形色色的發行版

有些組織或廠商將多種軟體組合起來，與 Linux 系統核心一併包裝、發行或販賣，成為現在為數眾多的各種 Linux 發行版 (distribution)。
發行版特色
Debian GNU/Linux社群驅動，穩定優先
Ubuntu簡單好用的人性化介面
Fedora由 Red Hat 贊助，創新優先
openSUSE企業級應用導向
CentOS企業級 Linux 社群版
Linux Mint易於使用，桌面導向

📌 1-2 核心 (Kernel) 與發行版的關係

1
核心 (Kernel)
作業系統最重要的心臟部位。負責所有讓系統得以正確、有效運行相關的工作。
2
發行版 (Distribution)
將多種軟體（網頁瀏覽器、文書編輯、影像處理、辦公室軟體、程式開發工具、架站軟體等）與 Linux 系統核心一併包裝的完整作業系統。

📌 1-3 Linux 的用途

🖥️超級電腦
🌐伺服器
📱行動裝置（Android）
🔧嵌入式裝置
一應俱全的功能：網路瀏覽器、文書編輯、影像處理、辦公室軟體、程式開發工具，甚至架設伺服器等等的軟體。
從物聯網到超級電腦的廣泛用途。

💼 2｜為什麼要學習 Linux？

大公司的 IT 部門多半是採用 Windows 與 Linux 混合使用的環境，建議要有基礎的 Linux 知識。
除了嵌入式系統、伺服器等需要 Linux，許多 IT 領域需要的軟體只能在 Linux 環境運行，或是在 Linux 環境能有最好的執行效率。
💡 學習 Linux 的理由
  • 產業剛需：伺服器、嵌入式、雲端運算的主流環境
  • 效率優勢：許多開發工具在 Linux 環境執行效率最佳
  • 就業競爭力：IT 工程師的基礎能力之一

⌨️ 3｜習慣文字模式的操作

Linux 發行版分為純文字模式及圖形模式。
但各個雲端服務所提供的 Linux 虛擬機器預設沒有提供圖形模式。
此外，Linux 為系統的嵌入式裝置或網路通訊設備在開發、測試階段都是使用遠端連線進文字模式來設定。
⌨️ 文字模式的優點：不管換到什麼 Linux 發行版都很容易上手。
