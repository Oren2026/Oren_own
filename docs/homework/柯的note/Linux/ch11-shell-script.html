📜 1｜Shell Script 簡介

🐚 Shell Script 的定義

Shell Script 是利用 bash 等 Shell 所提供的語法撰寫的腳本，具備接受指令列參數、使用者輸入、輸出訊息及設定變數的能力。

本章以 bash shell script 為主。Script 中的指令可分為：

  • Linux 指令：如 ls、cp、grep
  • Shell 指令：如 if、for、while、read

📜 2｜建立與執行 script

1
方法一：傳給 shell 程式
bash script檔名 或 sh script檔名
2
方法二：直接執行
設定執行權限：chmod +x script檔名，再直接輸入檔名執行

⚠️ 若直接執行出現「找不到指令」錯誤，執行 export PATH=$PATH:. 或將路徑加入 ~/.bash_profile。

📜 3｜變數設定與取用

🔤 變數基本操作
  • 設定變數：VAR=value（等號兩側不能有空格）
  • 取用變數：$VAR 或 ${VAR}
  • 顯示脫逸 $：使用 \$ 或單引號包圍
🔄 指令置換

將指令輸出結果設為變數值，有兩種語法：

  • 反引號：VAR=\`command\`
  • $( )：VAR=$(command)（推薦，語意更清晰）
🔢 指令列參數

執行時傳入參數，script 內以 $1、$2... 取用，$# 為參數總數，$* 或 $@ 為所有參數。

📜 4｜環境變數與 read 輸入

系統中有許多內建環境變數可直接取用，如 $HOME、$PATH、$USER、$PWD。

⌨️ read 互動輸入

使用 read 變數名稱 可暫停 script 執行，等待使用者輸入並存入變數。

📜 5｜test 指令與運算子

✅ test 的用途

test 用來判斷運算式的真假，常用於 if、while 等流程控制的條件測試。另一種等價寫法：[ 運算式 ]

類型運算子說明
字串= / ==兩字串相等
!=兩字串不相等
-z字串為空
-n字串不為空
-n字串存在且長度 > 0
數值-eq相等
-ne不相等
-lt / -le小於 / 小於等於
-gt / -ge大於 / 大於等於
檔案-e檔案存在
-f一般檔案存在
-d目錄存在
-r / -w / -x可讀 / 可寫 / 可執行
-L為符號連結
-s檔案存在且不為空
邏輯-aAND（兩者皆真）
-oOR（兩者之一為真）
!NOT（取反）

📜 6｜流程控制 if / case

🔀 if 敘述

根據條件真假決定執行的程式段落。結構：if ... then ... elif ... then ... else ... fi

🔀 case 敘述

從多個條件中選擇第一個符合的執行，適合字串比對。;; 為結束符號，*) 為預設執行。

⚠️ 使用萬用字元作為測試條件時，請勿於字串兩側加上雙引號。

📜 7｜迴圈 for / while / until

類型語法說明
for（串列）for VAR in 串列; do ... done對串列中每個元素執行一次
for（C風格）for ((i=0; i<10; i++)); do ... done類C語法，支援數值運算
whilewhile [ 條件 ]; do ... done條件為「真」時重複執行
untiluntil [ 條件 ]; do ... done條件為「假」時重複執行

break：中斷迴圈；continue：跳過本次迭代。可加數字參數跳脫多層迴圈。

📜 8｜shift 參數移位

⬅️ shift 的作用

shift 將指令列參數向左移動一位，使 $2 變成 $1、$3 變成 $2，常用於逐一處理未知數量的參數。

shift（預設左移1位）、shift 2（左移2位）。常與 while 迴圈合用：while [ "$#" -gt 0 ]; do ...; shift; done

📜 9｜函式定義

🧩 函式語法

當 script 變大時，可將常用指令寫成函式以便重複呼叫。

取用函式回傳值：直接取用全域變數（如 $maxvalue），或使用 return 搭配 $?。

📜 10｜export 與 source

指令說明
bash script.sh系統產生新的 shell 執行，script 內的變數不會影響目前 shell
source script.sh以目前 shell 環境執行，script 內的變數會保留下來（相當於 . 指令）
export VAR將變數輸出為環境變數，供其他程式使用

💡 若要讓 script 中的設定永久生效，使用 source 而非直接執行。
