📦 1｜打包壓縮簡介

🗜️ 為什麼要打包壓縮？

Linux 大部分程式以壓縮檔形式散佈，常見副檔名包括 .tar.gz、.tgz、.gz、.bz2、.zip。

流程：打包（tar）先將多個檔案合併為一個 → 壓縮再縮小體積。

📦 2｜tar 指令

參數說明
-c建立新的 tar 檔（create）
-x解開 tar 檔（extract）
-t檢視壓縮檔內容（list）
-v顯示運作過程資訊（verbose）
-f 檔名指定檔案名稱
-z呼叫 gzip 壓縮/解壓（.tar.gz）
-j呼叫 bzip2 壓縮/解壓（.tar.bz2）
-Z呼叫 compress 壓縮/解壓（.tar.Z）

📦 3｜tar 進階操作

參數說明
--atime-preserve解開時保留原始檔案時間
-h打包時保留符號連結所指向的檔案
-k解開時不覆蓋已存在的檔案
--totals打包完成後顯示寫入的位元組數
--delete刪除 tar 檔內的特定檔案
-r將檔案附加到現有 tar 檔
--remove-files打包後移除原始檔案

常見用法：

解開 .tar.gz：tar -xzvf 檔案.tar.gz

建立 .tar.gz：tar -czvf 檔案.tar.gz 目錄/

解開 .tar.bz2：tar -xjvf 檔案.tar.bz2

📦 4｜zip / unzip

指令說明
zip 檔名.zip 檔案...壓縮指定檔案
zip -g 檔名.zip 檔案將檔案加入已存在的 zip 檔
zip -j 檔名.zip 檔案只儲存檔案名稱，不含目錄結構
zip -r 檔名.zip 目錄遞迴壓縮，包含所有子目錄
unzip 檔名.zip全部解壓縮至目前目錄
unzip 檔名.zip -d 目錄解壓縮至指定目錄
unzip -o 檔名.zip若檔案已存在則覆蓋
unzip -n 檔名.zip僅解壓縮不存在的檔案

📦 5｜gzip / gunzip

gzip 只能壓縮單一檔案，無法處理目錄（需先以 tar 打包）。

常用指令：

gzip -h：顯示說明

gzip 檔案：壓縮（自動產生 .gz 檔，原始檔案刪除）

gunzip 檔案.gz 或 gzip -d 檔案.gz：解壓縮

gzip -f：強迫覆蓋已存在的輸出檔，不詢問

📦 6｜bzip2 / bunzip2

🗜️ bzip2 的特性

bzip2 與 bunzip2 在功能與參數用法上與 gzip 幾乎相同，但壓縮率通常比 gzip 更高，適合需要節省空間的場景。

解開 .tar.bz2：tar -xjvf 檔案.tar.bz2

或分開處理：bzip2 -d 檔案.tar.bz2 → tar -xf 檔案.tar
