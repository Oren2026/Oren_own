🤖 1｜機器學習環境簡介

🧠 機器學習 Framework

TensorFlow（Google）和 PyTorch（Meta）是目前最主流的機器學習框架，能幫助開發者快速建立與部署 ML 模型。

⚡ GPU 加速的需求

機器學習通常需要 GPU 的運算能力，因此需安裝 Nvidia 顯示卡驅動程式及設定 CUDA 開發環境。

🤖 2｜CUDA / cuDNN 安裝

🔧 CUDA 是什麼

CUDA（Compute Unified Device Architecture）是 Nvidia 的平行運算平台，讓開發者能用 GPU 進行圖形以外的通用計算。cuDNN 則是專為深度學習設計的 GPU 加速庫。

CUDA 下載：https://developer.nvidia.com/cuda-downloads

cuDNN 下載：https://developer.nvidia.com/cudnn

🤖 3｜Nvidia 驅動與環境設定

步驟指令 / 動作
新增驅動儲藏庫新增 PPA 後以 apt 安裝 Nvidia 驅動
驗證驅動nvidia-settings 或 nvidia-smi
設定環境變數於 /etc/profile 加入 CUDA 路徑
設定函式庫路徑於 /etc/ld.so.conf 加入 CUDA lib，執行 sudo ldconfig
驗證 CUDA編譯並執行 CUDA 範例（如 deviceQuery）

cuDNN 安裝後，額外的 deb 套件會放在 /var/cudnn-local-repo-* 目錄，需手動以 dpkg -i 安裝。

🤖 4｜Anaconda 安裝

📦 Anaconda 的用途

Anaconda 內建數百個機器學習、科學運算相關的 Python 套件，是管理 ML 環境的最佳工具。

下載：https://www.anaconda.com/products/distribution

安裝：bash Anaconda3-版本.sh

安裝完成後需重新登入或開啟新終端機，Anaconda 的 base 環境會自動啟動。

🤖 5｜conda 環境管理

指令說明
conda create -n 名稱 python=版本建立虛擬環境（如 conda create -n TensorFlow python=3.9）
conda activate 名稱啟動環境
conda deactivate離開環境
conda env list列出所有環境
conda config --set auto_activate_base false關閉開機自動啟動 base 環境

🤖 6｜TensorFlow 安裝

1
建立環境
conda create -n TensorFlow python=3.9 && conda activate TensorFlow
2
安裝 TensorFlow
pip install tensorflow-gpu==2.9.1（GPU 版）或 pip install tensorflow（CPU 版）
3
驗證安裝
以 Python 測試 import tensorflow as tf; print(tf.__version__)

🤖 7｜PyTorch 安裝

PyTorch 官網提供互動式安裝指令產生器，可根據 GPU/CPU 版本給出正確指令。

GPU 版：conda install pytorch torchvision torchaudio cudatoolkit=11.6 -c pytorch -c conda-forge

CPU 版：conda install pytorch torchvision torchaudio cpuonly -c pytorch

驗證：import torch; print(torch.cuda.is_available())（若輸出 True 表示 GPU 版安裝成功）
