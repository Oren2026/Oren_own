📝 1｜nano 編輯器

✏️ nano 特色

nano 是 Ubuntu 內建的文書編輯軟體，使用者介面相當方便，即使是第一次使用的人都能很快熟悉。nano 的操作方式類似 GUI 編輯器，底部有常用操作的快速鍵提示。

快速鍵功能
^G（Ctrl+G）顯示輔助說明
^O儲存檔案（WriteOut）
^R插入其他檔案內容
^Y向上一頁（PageUp）
^V向下一頁（PageDown）
^K剪下整行文字
^U貼上（還原剪下）
^C顯示目前游標位置
^X結束 nano（若有未儲存修改，會提示）

啟動：nano（空白檔案）或 nano 檔案名稱（開啟指定檔案）

📝 2｜vim 三種模式

💡 vim 簡介

vim（Vi IMproved）是功能強大的文字編輯器，安裝方式：sudo apt install vim。vim 的核心概念在於「模式」——不同模式下同一按鍵代表不同意義。

模式進入方式可做什麼離開方式
Normal Mode
（一般模式）		 vim 啟動後預設模式 無法輸入文字，但可用按鍵指令執行刪除、複製、貼上、游標移動等操作 按 i、a、o 進入 Insert Mode；按 : 進入 Command-line Mode
Insert Mode
（插入模式）		 在 Normal Mode 下按
i（插入）、a（附加）、o（新增一行）		 正常輸入文字內容 按 Esc 回到 Normal Mode
Command-line Mode
（命令列模式）		 在 Normal Mode 下按 : 儲存、搜尋、取代、結束等高階操作 按 Esc 回到 Normal Mode

進入 Insert Mode 的三種方式：

  • i：從游標所在位置插入新輸入的字元
  • a：從游標所在位置的下一個字元開始輸入
  • o：新增加一行，並將游標移到新行的開頭

📝 3｜vim 開啟、編輯與儲存

動作Command-line 指令說明
開啟 vimvim開啟空白編輯器
開啟檔案vim 檔案名稱開啟指定檔案，無此檔案則新建
儲存:w 檔案路徑及名稱寫入磁碟（可指定路徑）
開啟舊檔:e 檔案路徑及名稱在 vim 內開啟另一個檔案
結束:q離開（若有未儲存修改則失敗）
強制結束:q!強制離開，不儲存修改
儲存並離開:wq 或 :x寫入後關閉

標準工作流程：vim 檔案 → 按 i 進入 Insert Mode 打字 → 按 Esc 回 Normal Mode → :wq 儲存離開。
