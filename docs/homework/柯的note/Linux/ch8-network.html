🌐 1｜安裝網路套件

Ubuntu 預設沒有安裝 ifconfig 指令（用於檢視所有網路介面狀態）。需先安裝：

sudo apt install net-tools

🌐 2｜xDSL 寬頻撥接

📡 非固定制（計時制）寬頻

Linux 支援 xDSL 寬頻上網。本節介紹非固定制（計時制、撥接制）的設定方式。

1
安裝 PPPoE 套件
sudo apt install pppoeconf
2
執行設定精靈
執行 sudo pppoeconf，依指示輸入帳號密碼
3
連線與斷線
sudo pon dsl-provider 連線，sudo poff 斷線

驗證連線：執行網路狀態檢查指令，確認已成功取得 IP。

🌐 3｜固定 IP 設定（nmcli）

1
查詢網路介面
nmcli connection show 或 nmcli device status
2
設定 IP 與網路遮罩
nmcli connection modify "enp3s0" ipv4.addresses 10.10.10.100/8
3
設定閘道器
nmcli connection modify "enp3s0" ipv4.gateway 10.10.10.1
4
設定 DNS
nmcli connection modify "enp3s0" ipv4.dns "8.8.8.8"
5
套用並啟動
nmcli connection up "enp3s0"

驗證：ifconfig 與 route -n 檢查 IP、網路遮罩、預設閘道是否正確；nmcli device show 檢查完整設定。

🌐 4｜DHCP 自動取得 IP

🔄 DHCP 運作流程

DHCP（Dynamic Host Configuration Protocol）可讓網路介面自動向路由器取得 IP 位址、子網路遮罩、預設閘道、DNS 等資訊，無需手動設定。

將固定 IP 改為 DHCP：

nmcli connection modify "enp3s0" ipv4.method auto

nmcli connection up "enp3s0"

🌐 5｜無線網路設定

1
確認網卡驅動
檢查系統是否正確驅動無線網路卡
2
掃描基地台
nmcli device wifi list 列出可用無線網路
3
連線
nmcli device wifi connect "SSID名稱" password "密碼"
4
驗證
ifconfig 確認網卡啟動且 IP 正確；iwconfig 檢視無線資訊

若網卡未啟動：sudo ip link set wlan0 up 或使用 nmcli networking on。

🌐 6｜網路偵錯工具

指令用途範例
ping IP/網域測試區域網路或對外連線是否可達ping 8.8.8.8、ping google.com
route -n顯示路由表，確認預設閘道設定檢查 0.0.0.0 閘道是否正確
nslookup 網域查詢 DNS 解析結果nslookup google.com
ip addr show顯示所有網路介面的 IP 位址確認各介面已取得正確 IP
dhclient向 DHCP 伺服器重新請求 IPsudo dhclient -r && sudo dhclient

常見偵錯流程：ping 區域網路閘道（確認區網）→ ping 8.8.8.8（確認對外）→ ping google.com（確認 DNS）。若 DNS 解析失敗，檢查 /etc/resolv.conf 或 nmcli 中的 DNS 設定。
