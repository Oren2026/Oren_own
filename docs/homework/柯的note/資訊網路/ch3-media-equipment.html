🔌 1｜網路傳輸媒介

傳輸媒體（Transmission Media）
▶ 有線傳輸：藉電纜線、光纖等傳輸
▶ 無線傳輸（Wireless）：運用廣闊天際，無空間限制，支援移動性（Mobility）
▶ 目標：以最低成本降低傳輸衰減（Attenuation）率
▶ 一般網路傳輸仍以有線傳輸為主

🔀 2｜雙絞線（Twisted Pair）

為何要相互絞在一起？
減少電磁干擾（EMI）與串音（Crosstalk），提升訊號品質。
UTP — Unshielded Twisted Pair（無遮蔽雙絞線）
▶ 最常見、價格便宜、使用廣泛　▶ 僅有絞線本身，無額外金屬遮蔽層
STP — Shielded Twisted Pair（遮蔽雙絞線）
▶ 每對絞線有金屬遮蔽層，抗干擾能力更強　▶ 成本較高，適合工業環境

📡 3｜同軸電纜線（Coaxial Cable）

結構：中心導線（內層銅芯）+ 絕緣層 + 金屬遮蔽網 + 外皮
▶ 早年有線電視（Cable TV）和乙太網路（10Base2/10Base5）廣泛使用
▶ 抗干擾能力佳，傳輸距離比雙絞線更遠

🌈 4｜光纖（Fiber Optic Cable）

優點：重量輕、體積小、使用年限久、無幅射、無電磁干擾（EMI）、 低衰減、傳輸頻寬大、不受雷擊影響
結構：中心纖維（Core）→ 周邊材料（Cladding）→ 保護層（Protective Sheath）
光纜（Cable）：將許多細細光纖包裹在一起，方便佈線架設

📡 5｜無線電傳輸

🌊 地波（Ground Wave）
頻率：3 KHz ～ 3 MHz（極低頻，波長甚長）
傳輸：可達 150 公里
應用：航海通訊、AM 廣播
☁️ 天波（Sky Wave）
頻率：3 MHz ～ 30 MHz
機制：電離層導波通道
傳輸：150 ～ 400 公里
📏 直線傳送（LOS；Line-of-sight）
頻率：30 MHz 以上，直線行進、無法繞射
應用：無線電視、大哥大、衛星通訊、外太空
注意：基地台必須架設在至高點或高塔

🔗 6｜連接器（Connector）

RJ-45 與 RJ-11 連接頭
▶ RJ-45：用於乙太網路（8蕊）
▶ RJ-11：用於電話線（4～6蕊）
光纜連接頭
▶ LC 接頭：接頭較小，目前使用最普遍
▶ ST 接頭：採用於配線箱
▶ SC 接頭：集線器、網路元件器具採用

⚙️ 7｜網路傳輸設備

📡 數據機（Modem）
Modem = MOdulation + DEModulation
數位訊號 ↔ 類比訊號 相互轉換
🖥️ 網路卡（NIC）
Network Interface Card
不同網路類型有不同網路卡：有線網卡、USB 網卡、USB 無線網卡
🔄 中繼器（Repeater）
接收衰減訊號 → 重整（放大）→ 轉發
屬第一層（實體層）裝置
🔀 集線器（Hub）
多埠中繼器（Multi-port Repeater）
所有連接埠共享頻寬，屬第一層裝置，可與網管軟體搭配
🌉 橋接器（Bridge）
連接兩個相同類型網路區段，依 MAC 位址轉送資料，減少碰撞
屬第二層（資料連結層）裝置
🔀 交換式集線器（Switch）
又稱多埠橋接器（Multi-port Bridge）
各連接埠有獨立專屬傳輸頻寬，屬第二層裝置

Hub vs 交換式集線器 比較

項目集線器（Hub）交換式集線器（Switch）
層級第一層（實體層）第二層（資料連結層）
頻寬所有埠共享同一頻寬每個埠獨立專屬頻寬
傳輸廣播（所有埠都收到）點對點（依 MAC 位址轉送）
碰撞易碰撞，效率低碰撞減少，效率高
價格便宜較貴

路由器、Layer 3 交換器、Gateway

🧭 路由器（Router）
Route + Routing 的功能
連接多個網路，執行封包的路徑選擇，屬第三層（網路層）裝置
運用軟體方式進行處理
🔀 第三層交換器（Layer 3 Switch）
以硬體方式執行封包的尋徑工作，執行效能比軟體路由器更好
主要著重在封包路徑交換，架構單純簡單，價格反而比路由器便宜
支援 QoS、網路安全防護管理
Layer 3 Switch vs 路由器：
▶ Layer 3 Switch = 硬體尋徑 → 效能好、價格低
▶ 路由器 = 軟體尋徑 → 功能較多、相容性更廣
🚪 閘道器（Gateway）
▶ 具有比一般路由器更強的相容與轉換能力
▶ 只要 Gateway 有支援的架構，都可執行轉換，可連接不同通訊協定的網路
▶ 功能最強，相對價格也最高
🌐 廣域網路連接方式：
企業網路 ←→ 專線交換機 ←→ 廣域網路
常見廣域網路技術：專線、xDSL、光纖、MPLS

設備與 OSI 層級對照

OSI 層設備說明
第一層｜實體層中繼器、集線器放大訊號、廣播
第二層｜資料連結層橋接器、交換式集線器、第二層交換器依 MAC 位址轉送
第三層｜網路層路由器、第三層交換器依 IP 位址尋徑
應用層閘道器（Gateway）協定轉換、相容性最高
