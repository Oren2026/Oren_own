## 📌 採購原則

- **優先 3D 列印**：可自行設計的機構零件，優先使用 3D 列印製作
- **標準件採購**：無法自製或需要強度的零件（螺絲、軸承、金屬件等）另行採購
- **補助申請**：外島參賽補助最高 **15,000 元**，可列入預算規劃
