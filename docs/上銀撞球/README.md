## 📋 賽制

**小組賽 → 四強雙敗淘汰賽**

- 分為 A、B 兩組，各組前兩名晉級四強
- 四強賽採雙敗淘汰制
- 最終決出冠軍與亞軍，季軍由兩隊並列
