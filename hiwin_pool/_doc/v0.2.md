## 新架構

```
EventBus（事件匯流排）
    │
    ├── subscribe("vision.ball_position", callback)
    │         ↑______________↑______________↑
    │         │              │              │
    │    VisionPlugin   MotionPlugin   CommPlugin
    │         │              │              │
    │    (發佈frame)  (發佈arm_cmd)  (接收並執行)
    │
    └── subscribe("vision.frame", callback)
              ↑________↑
         VisionPlugin  UIPlugin
```

---

## 事件類型

| 事件 | 方向 | 資料格式 |
|------|------|---------|
| `vision.frame` | Vision → Bus | `numpy.ndarray` |
| `vision.ball_position` | Vision → Bus | `{"x", "y", "z", "confidence"}` |
| `motion.arm_command` | Motion → Bus | `{"x", "y", "z", "velocity"}` |
| `motion.strike_command` | Motion → Bus | `{"force"}` |
| `comm.arm_position` | Comm → Bus | `{"x", "y", "z", "a", "b", "c"}` |
| `comm.arduino_status` | Comm → Bus | `bool` |
| `ui.command` | UI → Bus | `"quit"` / `"strike"` |
