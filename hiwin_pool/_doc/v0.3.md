## 實作細節

### 演算法流程

```
frame (cv2.VideoCapture.read())
  ↓
BallDetector.detect(frame) → ball_pixel {ball_x, ball_y, found}
  ↓
found == True → pixel_to_table(ball_x, ball_y) → table_x, table_y
  ↓
confidence = min(1.0, area / 1000.0)
  ↓
BallPosition(x=table_x, y=table_y, confidence=confidence)
  ↓
bus.publish("vision.ball_position", ball_pos.to_dict())
```
