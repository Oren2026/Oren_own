## 介面 dataclass

```python
@dataclass
class ArmCommand:
    x: float       # 目標 X（mm）
    y: float       # 目標 Y（mm）
    z: float       # 目標 Z（mm）
    velocity: float  # 速度（mm/s）
    motion_type: str  # "LIN" 或 "PTP"

@dataclass
class StrikeCommand:
    force: float   # 力道 0.0 ~ 1.0（線性，非段數）

@dataclass
class ArmPosition:
    x: float
    y: float
    z: float
    a: float
    b: float
    c: float
```

## 實作細節

### TCP/IP（手臂）

- COPEN 建立連線（handle 管理）
- CWRITE 發送 HRL 指令（PTP / LIN）
- CREAD 讀取回應
- CCLEAR 關閉連線

### USB（Arduino）

- `S{force}\n` 發送力道（0.0 ~ 1.0 映射到步進馬達脈衝）
- `R\n` 復位
- `P\n` 查詢狀態
