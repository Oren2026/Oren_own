## 計算邏輯（規劃）

### 球檯座標 → 手臂座標

```
球檯原點（TABLE_ORIGIN）已知
手臂 home 位置已知
球檯 X/Y → 手臂 X/Y（需 homography 矩陣）

手臂 Z：
  - 移動時：SAFE_Z_HEIGHT（200mm）
  - 擊球時：STRIKE_Z_HEIGHT（50mm）
```

### 力道計算（線性）

```
force = f(球位置, 桿頭偏移, 擊球角度)
       = a * distance + b * angle_offset + c * spin
       → 輸出 0.0 ~ 1.0
```

### 擊球序列

```
1. 移動到球上方（safe height）
2. 水平對位
3. 下降到 strike height
4. 發送 strike_command（force）
5. 發送手臂到位指令（推桿）
6. 回 home
```
