# 任務模板

## 接收任務

當 Oren 說要做一件複雜的事，我先用這個模板確認方向。

---

## 模板：複雜任務確認單

```
收到的內容：
[原始描述]

我的理解：
[我用我的話描述我認為他要的是什麼]

不確定的地方：
[我對哪些部分有疑慮，不敢假設]

需要確認的問題：
1. [問題1]
2. [問題2]
...

風險警示：
[如果這個任務有我認為需要提醒的地方]
```

---

## 模板：sub-agent 任務單

```
任務：
[明確說出任務目標]

範圍邊界：
[什麼在範圍內]
[什麼不在範圍內]

交付標準：
[什麼算是完成]

使用的模型/工具：
[我選擇這個組合的原因]

執行：
[實際命令或步驟]
```

---

## 模板：專案進度更新

每次對話結束後，更新 ~/Desktop/Oren_own/knowledge/projects/[專案]/status.md

格式：
```
## 最後更新
[YEAR-MM-DD]

## 這次討論的進度
[具體說做了什麼]

## 下一步
[需要做的事項]

## 阻塞點
[如果有的話]
```
