單選Q6比較
相較於 Hub，Switch 能顯著提升網路效能的原因是？
✅ 正確答案：A
Switch 會學習 MAC 位址（CAM table），只把封包送到目標連接埠，不像 Hub 廣播給所有埠，減少碰撞並提升效能。

單選Q7MAC位址
MAC 位址 (Media Access Control Address) 的特性是？
✅ 正確答案：A
MAC 位址長度 48 位元（12 位十六進位），由製造商燒錄，全世界唯一（如 00:1A:2B:3C:4D:5E）。
B 描述的是 IP 位址（可變且在同一網段內唯一）；C 是 Router 功能；D 是 IP 位址（32 位元 IPv4）。
點擊題目展開答案
單選Q8比較
Hub 的廣播式轉發會導致什麼問題？
✅ 正確答案：A
Hub 將封包廣播給所有連接埠，所有節點共享頻寬，容易發生資料碰撞（Collision），浪費頻寬並造成網路效能下降。

關於 Layer 1（實體層）設備，以下何者正確？
✅ 正確答案：A
Hub 和 Repeater 屬於 Layer 1，僅處理電氣訊號，不做邏輯判斷。學習 MAC 位址是 Layer 2 Switch 的功能。

📶 第一章：網路導論

🌐 LAN vs WAN
LAN（區域網路）：範圍小、速率高、延遲低、自行管理
WAN（廣域網路）：範圍大、速率低、延遲高、由電信公司維運
🔷 網路拓撲結構
星狀 (Star)：中心交換機，中心故障全網癱瘓
環狀 (Ring)：封閉環中流動
匯流排 (Bus)：共用主幹，易碰撞
OSI 七層模型
📱 上三層（軟體介面）
L7應用層HTTP, FTP, SMTP, DNS
L6表現層加密, 壓縮, 格式轉換
L5會話層對話控制, API
🔧 下四層（傳輸連接）
L4傳輸層TCP, UDP, 流量控制
L3網路層IP, 路由選擇
L2資料連結層MAC, 框架, Switch
L1實體層電壓, 光纖, Hub

📡 第二章：通訊原理

📈 訊號類型
類比訊號：連續波形（如聲音、光線）
數位訊號：離散的 0 與 1，抗干擾能力強、易處理
📉 衰減 (Attenuation)
訊號隨傳輸距離增加而能量減弱
解決方案：中繼器 (Repeater) 放大訊號
📡 傳輸模式
📺
單工 Simplex
僅單向（廣播、電視）
📻
半雙工 Half-Duplex
雙向但不能同時（對講機）
📱
全雙工 Full-Duplex
同時雙向（手機通話）
⚡ 頻寬 (Bandwidth)
傳輸媒介的載流能力，直接影響網路速度
單位：bps（位元/秒）
頻寬越高 = 傳輸速度越快
🔄 Modem（調變解調器）
Mo = Modulation（調變）：數位→類比
Dem = Demodulation（解調）：類比→數位
讓電腦能透過電話線聯網

🔧 第三章：網路設備

Layer 1 — 實體層設備
設備功能特點
Hub廣播轉發所有埠收到，安全性低，易碰撞
Repeater訊號放大克服衰減，延長傳輸距離
Layer 2 — 資料連結層設備
設備功能特點
SwitchMAC 位址轉發學習 CAM table，精確轉發，減少碰撞
NIC網路介面唯一 MAC 位址（48位元，製造商燒錄）
Layer 3 — 網路層設備
設備功能特點
RouterIP 路由不同網路間路徑選擇，使用邏輯位址 (IP)
🔑 重要記憶口诀
Hub = 一個老師對全班廣播（L1）
Switch = 快遞員看地址精準送達（L2，MAC）
Router = 快遞公司跨城市轉運（L3，IP）
Repeater = 加油站，延長旅程（L1）
MAC vs IP 位址
MAC 位址：48位元，燒錄在網卡，製造商決定，唯一
IP 位址：32位元（IPv4），可由使用者配置，邏輯位址
