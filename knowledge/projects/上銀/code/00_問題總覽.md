# 上銀撞球 - 問題總覽

> 目的：發散列舉所有可能的問題與挑戰，之後收斂出可重用的模組。

---

## 1. 環境與場地

| 問題 | 說明 | 優先級 |
|------|------|--------|
| 光線變化 | 比賽現場光線可能影響視覺辨識 | 高 |
| 球檯平面水平 | 地面不平影響擊球軌跡 | 中 |
| 球卡在洞口 | 球停在洞口邊緣，進退兩難 | 中 |
| 洞口視角被擋 | 目標洞口視角被其他球遮擋 | 高 |

---

## 2. 視覺辨識

| 問題 | 說明 | 優先級 |
|------|------|--------|
| 雙目校正精度 | 兩顆鏡頭的校正準確度影響測距 | 高 |
| 球辨識準確率 | OpenCV 能否正確辨識所有球 | 高 |
| 洞口辨識 | 洞口位置偵測是否準確 | 高 |
| 即時追蹤延遲 | 從拍攝到計算輸出的延遲 | 中 |
| 球重疊判斷 | 兩球靠在一起時的辨識 | 中 |
| 白球VS彩球 | 白色球的辨識干擾 | 待確認 |

---

## 3. 擊球策略

### 3.1 基本擊球模式

| 模式 | 說明 | 優先級 |
|------|------|--------|
| 直接擊球 | 球桿直接對準目標球和洞口擊打 | 高 |
| 白球停位 | 撞擊後白球停在指定位置 | 高 |
| 轉向擊球 | 利用邊框反彈擊球 | 中 |
| 避障擊球 | 擊球路徑被擋，需繞路 | 高 |
| 薄球擊打 | 輕碰目標球邊緣 | 中 |
| 旋轉擊球 | 加側旋（桿頭旋轉） | 低 |

### 3.2 障礙應對策略

| 策略 | 觸發條件 | 優先級 |
|------|----------|--------|
| 原地擊球 | 擊球路徑無障礙 | 高 |
| 角度偏移 | 直線路徑被擋，偏移角度擊打 | 高 |
| 邊框反彈 | 偏移擊球也無法達成 | 中 |
| 放棄擊球 | 無法達成，通知人工介入 | 低 |

---

## 4. 機構與力道控制

| 問題 | 說明 | 優先級 |
|------|------|--------|
| 擊球力道對應 | 不同距離需要多少力道 | 高 |
| 力道控制精度 | 馬達輸出一致性 | 高 |
| 桿頭磨損 | 磨損後力道會改變 | 中 |
| 機構彈性 | 機構變形吸收力道 | 低 |
| 下桿時機 | 桿頭接觸球的最佳時機 | 待研究 |

---

## 5. 物理模型

| 項目 | 說明 | 優先級 |
|------|------|--------|
| 球與球碰撞 | 碰撞後兩球的分開角度 | 高 |
| 球與邊框碰撞 | 反彈角度計算 | 高 |
| 摩擦力 | 球滾動時的減速 | 中 |
| 旋轉效果 | 側旋對球軌跡的影響 | 低 |
| 白球後旋 | 撞擊後白球的旋轉 | 待研究 |

---

## 6. 通訊與控制

| 問題 | 說明 | 優先級 |
|------|------|--------|
| Modbus TCP 連線 | 手臂與上位系統的溝通 | 高 |
| 指令延遲 | 指令傳送到執行的延遲 | 中 |
| 異常處理 | 通訊中斷時的處理 | 中 |
| 同步問題 | 多軸同時運動的一致性 | 低 |

---

## 7. 比賽特定問題

| 問題 | 說明 | 優先級 |
|------|------|--------|
| 白球犯規 | 白球進洞或被撞飛 | 高 |
| 擊球次數限制 | 比賽有時間/次數限制 | 待確認 |
| 複審評分標準 | 影片評分的依據是什麼 | 待確認 |
| 球檯誤差 | 實際球檯與標準尺寸差異 | 待確認 |

---

## 模組收斂方向（初稿）

```
┌─────────────────────────────────────────────────────────┐
│  模組化建議                                              │
├─────────────────────────────────────────────────────────┤
│  M1: 視覺介面模組                                       │

│                                                         │
│  M2: 策略決策模組                                       │

│  M4: 力道控制模組                                       │
│      - 力道與馬達參數對應                               │
│      - 擊球時序控制                                     │
│                                                         │
│  M5: 手臂通訊模組                                       │
│      - Modbus TCP 指令封裝                              │
│      - 狀態監控                                         │
└─────────────────────────────────────────────────────────┘
```
