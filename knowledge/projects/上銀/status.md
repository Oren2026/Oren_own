## 比賽內容
- 第十八屆上銀智慧機器手實作競賽 — 撞球組
- 使用 HIWIN RA605-710-GC 六軸手臂
- 主辦單位提供手臂，參賽者需自行設計末端機構

## 手臂規格（RA605-GC）

### 法蘭規格（已確認）
| 參數 | 數值 |
|------|------|
| 中心孔 | Ø20mm（H7），深度 4mm |
| 定位銷孔 | Ø5mm（H7），深度 7mm，位於 12 點鐘方向 |
| 外部凸緣 | Ø40mm（h8），深度 6mm |
| 安裝螺孔 | 4 × M5，PCD Ø31.5mm |
| 螺紋深度 | 6mm |

### 通訊方式
- **Modbus TCP**（推薦，乙太網路）
- 可用 Ethernet TCP/IP 直接發 PTP/LIN 指令
- 支援 PR 位置暫存器（1~4000 組）
