## CCC作業天地 — HW_firstweb0327（網頁日誌系統）

### 作業規格（老師要求）
- JS + SQLite（sql.js）實作網頁日誌
- 不同版本有獨立的更新紀錄
- 版本間各自獨立可訪問

### 版本結構（最終）
```
HW_firstweb0327/
├── README.md          ← 三版本改版日誌總覽
├── v1.0/             ← 初版（基礎 CRUD，無使用者系統）
├── v1.1/             ← 第一正式版（Auth Modal + 公開日誌牆）
│   ├── index.html
│   ├── journal.js
│   ├── seed.db
│   └── README.md
└── v1.2/             ← 最新版（作者過濾功能）
    ├── index.html
    ├── journal.js
    ├── seed.db
    └── README.md
```

### 技術架構（v1.1 / v1.2）
- **sql.js** (WASM SQLite) + **IndexedDB** 持久化
- `seed.db` 預設資料庫（Node.js 建立，4人10篇文章）
- **SHA-256** 密碼雜湊（Web Crypto API）
- **sessionStorage** 管理登入 session

### v1.2 新功能（bcce09d）
- 點擊作者名稱 → 只看該作者文章
- 右上角 filter chip（👤 作者名 ×），點 ✕ 清除恢復全顯示
- 詳情 Modal 裡的作者也可以點

### DB Key 管理
- v1.1: `journal_sqlite_v2`（乾淨版本）
- v1.2: 共享同一 DB key

## ⚠️ 重要教訓：版本更新 SOP

**每次更新時**：
1. 複製當前最高版本資料夾（例如 `v1.2/` → `v1.3/`）
2. 在新資料夾中修改程式碼
3. 更新新資料夾的 `README.md`
4. 更新頂層 `HW_firstweb0327/README.md` 的改版日誌表格
5. commit 並 push
